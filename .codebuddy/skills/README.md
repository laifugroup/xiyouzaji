# Skills 导航目录

本目录包含《西游记》相关的AI技能文件。每个技能文件夹包含一个SKILL.md文件，提供了该技能的详细描述和使用指南。

## 可用技能

### [《西游记》章节设计技能](journey-to-the-west-chapter/SKILL.md)
- **描述**: 提供《西游记》小说章节设计，包括章节结构、情节单元、艺术特色、创作技巧等，适用于单章分析、教学讲解、创作模仿等场景。

### [《西游记》人物片段技能](journey-to-the-west-character/SKILL.md)
- **描述**: 提供《西游记》人物经典活泼/幽默片段及背景介绍，适用于写作参考、教育讲解、娱乐阅读等场景。

### [《西游记》大纲设计技能](journey-to-the-west-outline/SKILL.md)
- **描述**: 提供《西游记》小说大纲设计，包括整体情节概览、简单章节标题和2-3句章节总结，适用于快速了解故事框架、创作参考和教学入门。

### [《西游记》世界观技能](journey-to-the-west-world/SKILL.md)
- **描述**: 提供《西游记》世界观设定，包括三界体系、神仙等级、妖怪起源、法宝功能、地理设定、修炼体系等，适用于创作参考、文化研究、游戏设计等场景。

## 使用说明

每个SKILL.md文件包含了技能的完整定义，包括：
- 技能名称和描述
- 使用场景和示例
- 输入输出规范
- 相关资源和参考
