# 《西游记》人物片段技能

## 技能用途

此技能旨在提供《西游记》主要人物的经典活泼、幽默、生动片段，并附带详细的人物背景介绍和人物图谱分析。适用于：

- **写作参考**：为创作西游题材内容提供灵感与素材
- **教育讲解**：生动展示人物性格与故事桥段
- **娱乐阅读**：重温经典趣味情节
- **文化传播**：推广中国传统文学精髓
- **人物分析**：生成系统化的人物图谱和关系网络

## 何时使用此技能

当用户需要：
- 了解《西游记》某个人物的经典场景
- 获取生动的人物描写片段
- 查阅人物背景资料与性格分析
- 寻找写作灵感或教学素材
- 重温西游趣味故事

## 使用方法

### 1. 技能触发
用户提及"西游记"、"孙悟空"、"猪八戒"、"唐僧"、"沙僧"、"西游片段"、"人物背景"等相关关键词时，主动提供此技能。

### 2. 提供片段、背景与图谱
技能提供三种主要信息：
- **活泼/幽默片段**：经典场景的生动描述
- **背景介绍**：人物的出身、性格、重要经历、象征意义
- **人物图谱**：系统化的关系网络、成长轨迹、性格分析图谱

### 3. 检索参考资料
技能内包含以下参考资料，可按需查阅：

- [characters/README.md](references/characters/README.md)：人物资料库索引（点击查看详细人物介绍）
- [scenes.md](references/scenes.md)：经典片段库，按人物和场景类型分类
- [background.md](references/background.md)：西游记故事背景、文化意义、创作脉络
- [characters/CHECKLIST.md](references/characters/CHECKLIST.md)：完整人物清单，包含205个角色

### 4. 输出格式
默认提供结构化输出：

```markdown
## 人物：孙悟空

### 背景介绍
[人物背景、性格特点、重要经历]

### 经典活泼片段
[场景描述]

### 人物图谱
[系统化的人物关系、成长轨迹、性格分析图谱]
```

也可根据用户需求提供纯文本、JSON格式或完整叙述。

### 5. 生成人物图谱
当用户请求"生成人物图谱"或提及"人物图谱"时，提供以下结构化图谱：

- **基本信息图谱**：身份演变、特点分析
- **成长历程图谱**：从出身到修成正果的完整轨迹  
- **角色定位分析**：在团队中的位置和功能
- **性格特质图谱**：主要性格特点的量化分析
- **经典场景分析**：关键事件的时间轴
- **象征意义网络**：文化寓意和现代映射
- **人际关系图谱**：与其他角色的关系网络
- **现代启示映射**：原著品质到现代价值的转换

## 示例对话

**用户**："给我讲讲孙悟空的有趣故事"

**技能响应**：
```markdown
## 人物：孙悟空

### 背景介绍
孙悟空，又称齐天大圣、美猴王，由开天辟地以来的仙石孕育而生。他拜师菩提祖师，学会七十二变、筋斗云等高强法术。性格机智勇敢、调皮捣蛋，曾大闹天宫，后被如来佛祖压在五行山下五百年，经观音点化，保护唐僧西天取经。

### 经典活泼片段：大闹天宫偷仙桃
孙悟空被玉帝封为"弼马温"，后得知官职卑微，愤而返回花果山，自封"齐天大圣"。玉帝第二次招安，让他管理蟠桃园。这猴王进了蟠桃园，见那千年一熟、万年结果的仙桃，哪里忍得住？他支开土地、力士，自己爬上树梢，专拣那熟透的大桃，左一个右一个，吃得肚皮滚圆。吃饱了还不算，又在桃树上打盹，醒来再吃，把个蟠桃园祸害得七零八落。王母娘娘开蟠桃会时，七仙女来摘桃，发现树上只剩些青涩小桃，大桃全不见了踪影。

**性格体现**：顽皮贪吃、胆大包天、不守天规
```

**用户**："猪八戒有什么搞笑的情节？"

**技能响应**：
```markdown
## 人物：猪八戒

### 背景介绍
猪八戒原是天庭的天蓬元帅，因醉酒调戏嫦娥被贬下凡，误投猪胎。在高老庄被孙悟空降服，加入取经队伍。性格贪吃好色、懒惰耍滑，但也有憨厚可爱、关键时刻勇敢的一面。

### 经典活泼片段：高老庄招亲
猪八戒化身俊朗汉子到高老庄干活，一人能干十人的活，深得高太公喜欢，招为女婿。成亲当晚喝得大醉，现出原形——长嘴大耳、黑脸短毛的猪妖模样，把高家上下吓得魂飞魄散。他倒不伤人，只是把高小姐关在后院，每日好酒好菜供养。孙悟空变作高小姐模样戏弄他，八戒还憨憨地诉苦："娘子不知，我老猪虽模样丑，心里却实诚。若是你嫌我丑，等我取经回来，向佛祖讨个人模样便是。"

**性格体现**：憨厚实诚、好色贪杯、外貌自卑
```

2. **生动有趣**：强调片段的活泼幽默特质，避免枯燥叙述
3. **教育意义**：在娱乐中传递人物成长与人生哲理
4. **扩展性**：可根据用户反馈不断丰富片段库和人物资料
