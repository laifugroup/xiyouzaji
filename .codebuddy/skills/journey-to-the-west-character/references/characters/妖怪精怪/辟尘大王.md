# 辟尘大王

## 🐾 人物档案
- **姓名**：辟尘大王
- **类别**：妖怪精怪
- **身份**：金平府犀牛精三兄弟之一
- **特点**：能够辟尘，不畏风沙
- **能力**：精通风沙法术

## 📖 背景故事
辟尘大王是金平府的三只犀牛精之一，与辟寒大王、辟暑大王是兄弟。他们假扮佛像骗取香油，被孙悟空识破后展开激战。

## 🎭 性格特点
- **风沙法术**：掌握风沙相关的法术
- **配合默契**：与两个兄弟战术配合
- **狡猾贪婪**：善于伪装欺骗

## ✨ 经典片段

### 💨 风沙法术
```
辟尘大王施展法术，狂风大作："看我风沙之术！"
他掀起漫天风沙，遮蔽视线，展现强大的风沙法术。
```

### 🎭 三兄弟合击
```
辟尘大王与两个兄弟配合："我们三兄弟同心，何愁不能取胜！"
他们寒、热、风三种法术配合，让对手难以招架。
```

## 🤝 人物关系
- **兄弟**：辟寒大王、辟暑大王
- **对手**：孙悟空、四木禽星
- **受害者**：金平府百姓

## 🎨 文化象征
辟尘大王象征着风沙与变幻，体现了自然力量的不可预测性。

## 💡 现代启示
兄弟同心其利断金，但用错了方向反而会招致灾祸。
