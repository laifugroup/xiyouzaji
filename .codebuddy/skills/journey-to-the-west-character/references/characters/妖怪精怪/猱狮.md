# 猱狮

## 📋 人物档案
- **姓名**：猱狮
- **身份**：竹节山七狮之一，黄狮精的结拜兄弟
- **登场回目**：第九十回《师狮授受同归一 盗道缠禅静九灵》

## 📖 背景故事
猱狮是竹节山七狮之一，与黄狮精、雪狮、狻猊、白泽、伏狸、抟象结为兄弟。当黄狮精的兵器被孙悟空等人抢回后，七狮一起为黄狮精报仇。

猱狮作为七狮之一，法力较高，与其他六狮配合默契。但在与孙悟空等人的战斗中，七狮最终被消灭。

## 🎭 经典片段

### 1. **七狮结义**
猱狮与其他六狮结拜：

```
黄狮精："兄弟们，我们七狮结义，同心协力！"
猱狮："大哥说得对！有我们七兄弟在，天下无敌！"
雪狮："谁要是欺负我们兄弟，决不轻饶！"
```

### 2. **为兄报仇**
七狮为黄狮精报仇：

```
黄狮精："兄弟们，我的兵器被孙猴子抢走了！"
猱狮："什么？敢欺负我们大哥！兄弟们，为他报仇！"
七狮："为大哥报仇！找孙猴子算账！"
```

### 3. **七狮大战**
七狮与孙悟空等人激战：

```
猱狮："孙猴子！敢欺负我们大哥，看招！"
孙悟空："哈哈！七个狮子精，正好一网打尽！"
七狮："布阵！让这猴子尝尝我们的厉害！"
```

### 4. **被消灭**
七狮最终被孙悟空等人消灭：

```
孙悟空："八戒、沙师弟，我们分头对付！"
猪八戒："好嘞！看我的九齿钉耙！"
沙僧："我也来帮忙！"
（经过激战，七狮被消灭）
```

## 💡 性格特点

### 优点
- **重情重义**：为兄弟两肋插刀
- **团结协作**：与其他六狮配合默契
- **勇敢无畏**：敢于挑战强敌

### 缺点
- **盲目义气**：不问是非只为兄弟
- **实力不足**：虽然团结但实力不如孙悟空
- **缺乏判断**：没有判断事情的是非对错

## 🔗 人物关系
- **黄狮精**：结拜大哥
- **其他六狮**：结拜兄弟
- **孙悟空**：敌对关系

## 🌟 文化象征
猱狮代表了"兄弟义气"的形象：
- **江湖义气**：传统中的兄弟情义
- **团结力量**：团结就是力量
- **盲目忠诚**：有时义气会让人失去判断

## 💭 现代启示
1. **重视友情**：真正的友情值得珍惜
2. **明辨是非**：友情也要建立在是非基础上
3. **团结合作**：团结合作能产生更大力量

猱狮的故事展现了传统江湖义气的两面性。
