# 青毛狮子怪

## 📋 人物档案
- **姓名**：青毛狮子怪
- **身份**：文殊菩萨坐骑，乌鸡国假国王，狮驼岭三大王之一
- **登场回目**：第三十九回《一粒金丹天上得 三年故主世间生》

## 📖 背景故事
青毛狮子怪是文殊菩萨的坐骑，曾两次下界为妖。第一次是在乌鸡国，他变化成道士求雨成功，与国王结为兄弟，后将其推入井中淹死，自己变作国王统治乌鸡国三年。

第二次是在狮驼岭，他与黄牙老象、大鹏金翅雕结为兄弟，占据狮驼岭为妖。青毛狮子怪法力高强，曾一口吞下十万天兵。

## 🎭 经典片段

### 1. **乌鸡国为妖**
青毛狮子怪在乌鸡国变化成道士：

```
青毛狮子："陛下，贫道能求雨解旱。"
乌鸡国王："真的吗？若能求雨，朕与你结为兄弟！"
青毛狮子："陛下请看！"（成功求雨）
国王："神仙！真是神仙！我们结为兄弟！"
```

### 2. **害死国王**
青毛狮子怪将国王推入井中：

```
青毛狮子："兄弟，我们去御花园散步吧。"
国王："好！正好欣赏园中景色。"
青毛狮子："你去井边看看！"（将国王推入井中）
国王："啊！你..."（淹死）
```

### 3. **狮驼岭为王**
青毛狮子怪在狮驼岭称王：

```
青毛狮子："二弟、三弟，我们占据这狮驼岭，何等快活！"
黄牙老象："大哥说得对！有我们三兄弟在，谁敢来犯？"
大鹏金翅雕："就是如来佛祖来了也不怕！"
```

### 4. **吞十万天兵**
青毛狮子怪展示惊人本领：

```
天兵天将："妖怪！快束手就擒！"
青毛狮子："哈哈！让你们尝尝我的厉害！"
（张开大口，吞下十万天兵）
天兵："啊！这妖怪太厉害了！"
```

### 5. **被收服**
文殊菩萨前来收服青毛狮子怪：

```
文殊菩萨："孽畜！还不现出原形！"
青毛狮子："菩萨饶命！"（现出狮子原形）
文殊菩萨："随我回灵山受罚！"
```

## 💡 性格特点

### 优点
- **法力高强**：能一口吞下十万天兵
- **变化多端**：能变化成各种形态
- **领导能力**：在狮驼岭为三大王之首

### 缺点
- **心狠手辣**：害死乌鸡国国王
- **狂妄自大**：连天兵天将都不放在眼里
- **反复为恶**：两次下界为妖

## 🔗 人物关系
- **文殊菩萨**：主人
- **黄牙老象**：结拜兄弟
- **大鹏金翅雕**：结拜兄弟
- **乌鸡国国王**：害死的对象

## 🌟 文化象征
青毛狮子怪代表了"管教不严"的后果：
- **坐骑为患**：神仙坐骑下界为害
- **权力滥用**：拥有力量容易滥用
- **因果报应**：作恶多端终受惩罚

## 💭 现代启示
1. **严于律己**：拥有能力要严于律己
2. **尊重生命**：任何生命都值得尊重
3. **承担责任**：为自己的行为承担责任

青毛狮子怪的故事警示我们，即使出身高贵，如果缺乏约束，也可能危害他人。
