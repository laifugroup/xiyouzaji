# 黄狮精

## 📋 人物档案
- **姓名**：黄狮精
- **身份**：豹头山虎口洞妖怪
- **登场回目**：第八十九回《黄狮精虚设钉钯宴 金木土计闹豹头山》

## 📖 背景故事
黄狮精是豹头山虎口洞的狮子精，他偷走了孙悟空、猪八戒、沙僧的兵器，准备举办"钉钯宴"。与其他妖怪不同，黄狮精相对"文明"，他偷了兵器后还付钱给卖肉的，被称为"最有礼貌的妖怪"。

当孙悟空等人发现兵器被偷后，前来讨要。黄狮精请来祖翁九灵元圣相助，但最终被孙悟空等人打败。

## 🎭 经典片段

### 1. **偷取兵器**
黄狮精偷走三人的兵器：

```
黄狮精："好兵器！我要举办钉钯宴，请祖翁来观赏！"
小妖："大王，这些兵器真漂亮！"
黄狮精："快去准备宴席，我要大摆筵席！"
```

### 2. **"文明"妖怪**
黄狮精偷东西还付钱：

```
卖肉人："大王，你要的肉准备好了。"
黄狮精："这是银子，你收好。"
卖肉人："谢谢大王！您真是讲道理的妖怪！"
```

### 3. **举办钉钯宴**
黄狮精准备举办宴会：

```
黄狮精："小的们，把兵器摆好，我要请祖翁来观赏！"
小妖："大王，宴席都准备好了！"
黄狮精："好！快去请祖翁！"
```

### 4. **被讨要兵器**
孙悟空前来讨要兵器：

```
孙悟空："妖怪！快把我们的兵器还来！"
黄狮精："泼猴！兵器是我捡的，凭什么还你？"
孙悟空："偷了东西还有理？看打！"
```

### 5. **请祖翁相助**
黄狮精请九灵元圣相助：

```
黄狮精："祖翁！孙猴子来抢兵器了！"
九灵元圣："孙儿莫怕，有祖翁在！"
```

## 💡 性格特点

### 优点
- **相对文明**：偷东西还付钱
- **重视礼仪**：举办宴会讲究排场
- **尊敬长辈**：对祖翁九灵元圣十分尊敬

### 缺点
- **偷窃行为**：虽然文明但还是偷窃
- **依赖他人**：遇到困难就找祖翁
- **实力一般**：法力不如孙悟空

## 🔗 人物关系
- **九灵元圣**：祖翁，保护者
- **孙悟空等**：敌对关系
- **豹头山小妖**：手下

## 🌟 文化象征
黄狮精代表了"文明妖怪"的形象：
- **盗亦有道**：即使是妖怪也有自己的规矩
- **尊老爱幼**：传统美德在妖怪中也有体现
- **相对温和**：相比其他妖怪较为温和

## 💭 现代启示
1. **遵守规则**：任何时候都要遵守基本规则
2. **尊敬长辈**：尊老爱幼是重要美德
3. **文明行事**：即使是竞争也要文明进行

黄狮精的故事展现了妖怪世界中也有"文明"的一面。
