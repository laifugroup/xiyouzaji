# 伏狸

## 🐾 人物档案
- **姓名**：伏狸
- **类别**：妖怪精怪
- **身份**：狮驼岭妖王之一
- **特点**：狸猫成精，灵活机敏
- **能力**：善于伏击，行动敏捷

## 📖 背景故事
伏狸是狮驼岭的妖王之一，与其他狮精兄弟共同占据狮驼岭。虽然名字中带有"狸"字，但实际上是狮精家族的一员，与其他狮精关系密切。

## 🎭 性格特点
- **灵活机敏**：行动敏捷，善于变化
- **善于伏击**：精通伏击战术，出其不意
- **团结合作**：与其他狮精兄弟配合默契

## ✨ 经典片段

### 🐱 伏狸出击
```
伏狸身形灵活，如同狸猫般敏捷："让我来会会这猴子！"
他迅速出击，动作迅猛，让对手难以防备。
```

### 🤝 兄弟配合
```
伏狸与其他狮精配合默契："大哥，我从侧面进攻，你正面牵制！"
他们兄弟同心，战术配合天衣无缝。
```

## 🤝 人物关系
- **兄弟**：黄狮精、雪狮、狻猊等
- **对手**：孙悟空、天兵天将
- **部下**：狮驼岭众小妖

## 🎨 文化象征
伏狸象征着灵活与机敏，体现了动物精怪在战斗中的智慧与技巧。

## 💡 现代启示
在团队合作中，不同特长的成员应该发挥各自优势，相互配合。
