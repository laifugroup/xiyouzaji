# 孤直公

## 🌳 人物档案
- **姓名**：孤直公
- **类别**：植物精怪
- **身份**：荆棘岭木仙庵的柏树精
- **特点**：柏树成精，品格孤直
- **象征**：正直不阿，孤高自守

## 📖 背景故事
孤直公是荆棘岭木仙庵的柏树精，修行千年而成精。他性格孤直，品格高尚，与其他树精一起在木仙庵修行，喜欢吟诗作对。

## 🎭 性格特点
- **性格孤直**：为人正直，不随波逐流
- **品格高尚**：具有柏树的高洁品格
- **学识渊博**：修行千年，知识丰富

## ✨ 经典片段

### 🌳 柏树显形
```
孤直公化身老者，神情严肃："老夫乃山中柏树，修行千年，最重品节。"
他的形象庄重威严，展现千年树精的风范。
```

### 🎭 孤直性格
```
孤直公正色道："我等虽为精怪，但也知礼义廉耻，绝不做伤天害理之事。"
他的话语中透露出正直不阿的品格。
```

## 🤝 人物关系
- **同伴**：劲节十八公、凌空子、拂云叟、杏仙、赤身鬼
- **朋友**：唐僧（学问交流）
- **修行伙伴**：荆棘岭众树精

## 🎨 文化象征
孤直公象征着柏树的高洁品格，体现了中国文化中对正直品格的推崇。

## 💡 现代启示
正直的品格是立身之本，无论身处何种境地都应坚守原则。
