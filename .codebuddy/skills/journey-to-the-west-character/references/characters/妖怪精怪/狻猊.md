# 狻猊

## 🐾 人物档案
- **姓名**：狻猊
- **类别**：妖怪精怪
- **身份**：狮驼岭妖王之一，传说中的神兽
- **特点**：形似狮子，凶猛异常
- **象征**：力量与威严

## 📖 背景故事
狻猊是传说中的神兽，在《西游记》中作为狮驼岭的妖王之一出现。与黄狮精、雪狮等兄弟共同占据狮驼岭，威震一方。

## 🎭 性格特点
- **威严霸气**：具有神兽的威严气势
- **勇猛善战**：战斗力强大，凶猛异常
- **兄弟情义**：与其他狮精团结一致

## ✨ 经典片段

### 🦁 狻猊现身
```
狻猊身形高大，气势磅礴，站在山巅俯视众妖："我兄弟九人，今日齐聚狮驼岭，谁敢来犯！"
他的声音如同雷霆，震慑四方，小妖们无不拜服。
```

### ⚔️ 大战天兵
```
狻猊面对天兵天将，毫不畏惧："我乃上古神兽，岂是尔等小辈能敌！"
他凶猛异常，与天兵激战，展现神兽之威。
```

## 🤝 人物关系
- **兄弟**：黄狮精、雪狮、猱狮等
- **对手**：孙悟空、天庭众神
- **部下**：狮驼岭众妖

## 🎨 文化象征
狻猊在中国传统文化中是龙生九子之一，象征着威严与力量，体现了古代神话中神兽的形象。

## 💡 现代启示
真正的力量来自于内心的强大，而非外在的凶猛。
