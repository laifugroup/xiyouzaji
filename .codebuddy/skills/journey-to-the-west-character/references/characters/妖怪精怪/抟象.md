# 抟象

## 🐾 人物档案
- **姓名**：抟象
- **类别**：妖怪精怪
- **身份**：狮驼岭妖王之一
- **特点**：力量巨大，能搏斗大象
- **象征**：力量与勇猛

## 📖 背景故事
抟象是狮驼岭的妖王之一，与其他狮精兄弟共同占据狮驼岭。其名字"抟象"意为能够搏斗大象，体现了他的强大力量。

## 🎭 性格特点
- **力大无穷**：拥有搏斗大象的惊人力量
- **勇猛无畏**：面对强敌毫不退缩
- **兄弟情深**：与其他狮精团结一致

## ✨ 经典片段

### 💪 抟象显威
```
抟象大喝一声，声震山谷："让我来会会你这猴子！"
他力大无穷，一拳击出，山石崩裂，展现惊人的力量。
```

### 🐘 搏斗之威
```
抟象面对强敌，毫不畏惧："我连大象都能搏斗，何惧你这猴子！"
他的勇猛让对手也不得不认真对待。
```

## 🤝 人物关系
- **兄弟**：黄狮精、雪狮、狻猊等
- **对手**：孙悟空、天兵天将
- **部下**：狮驼岭众小妖

## 🎨 文化象征
抟象象征着极致的力量与勇猛，体现了古代对强大力量的崇拜。

## 💡 现代启示
真正的力量不仅在于肌肉，更在于内心的勇气与决心。
