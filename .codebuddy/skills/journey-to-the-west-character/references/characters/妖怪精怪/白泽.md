# 白泽

## 🐾 人物档案
- **姓名**：白泽
- **类别**：妖怪精怪
- **身份**：狮驼岭妖王之一，传说中的神兽
- **特点**：通晓万物，能言人语
- **象征**：智慧与知识

## 📖 背景故事
白泽是中国古代神话中的神兽，据说能够通晓万物之情，能说人话。在《西游记》中，白泽作为狮驼岭的妖王之一出现，与其他狮精兄弟共同占据狮驼岭。

## 🎭 性格特点
- **智慧超群**：通晓万物，知识渊博
- **能言善辩**：能够说人话，善于交流
- **冷静沉着**：相比于其他狮精更为理智

## ✨ 经典片段

### 🧠 白泽献计
```
白泽对众兄弟说道："孙悟空虽然厉害，但我们兄弟九人同心，何愁不能取胜？"
他的话语充满智慧，为众妖出谋划策，展现神兽的聪明才智。
```

### 🗣️ 与孙悟空对话
```
白泽面对孙悟空，不慌不忙："大圣，你虽然神通广大，但须知天外有天，人外有人。"
他的话语中带着哲理，让孙悟空也不得不认真对待。
```

## 🤝 人物关系
- **兄弟**：黄狮精、雪狮、狻猊等
- **对手**：孙悟空
- **智慧象征**：古代神话中的智者形象

## 🎨 文化象征
白泽在中国传统文化中是祥瑞之兽，象征着智慧与知识，据说黄帝曾向白泽询问鬼神之事。

## 💡 现代启示
智慧比力量更重要，真正的强大来自于知识和智慧。
