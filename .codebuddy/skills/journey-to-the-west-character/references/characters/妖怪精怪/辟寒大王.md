# 辟寒大王

## 🐾 人物档案
- **姓名**：辟寒大王
- **类别**：妖怪精怪
- **身份**：金平府犀牛精三兄弟之首
- **特点**：能够辟寒，不畏寒冷
- **能力**：精通寒冷法术

## 📖 背景故事
辟寒大王是金平府的三只犀牛精之一，与辟暑大王、辟尘大王是兄弟。他们假扮佛像，骗取香油，被孙悟空识破后展开激战。

## 🎭 性格特点
- **狡猾奸诈**：善于伪装，骗取香油
- **精通法术**：掌握寒冷相关的法术
- **兄弟同心**：与两个弟弟配合默契

## ✨ 经典片段

### 🧊 寒冷法术
```
辟寒大王施展法术，寒气逼人："看我寒冷之术！"
他口中吐出寒冰，瞬间将周围冻结，展现强大的寒冷法术。
```

### 🎭 假扮佛像
```
辟寒大王假扮佛像，骗取百姓香油："虔诚供奉，必得保佑！"
他的伪装十分逼真，骗过了众多百姓。
```

## 🤝 人物关系
- **兄弟**：辟暑大王、辟尘大王
- **对手**：孙悟空、四木禽星
- **受害者**：金平府百姓

## 🎨 文化象征
辟寒大王象征着贪婪与欺骗，提醒人们不要被表面的假象所迷惑。

## 💡 现代启示
表面的伪装终会被识破，真诚才是长久之道。
