# 杏仙

## 🌸 人物档案
- **姓名**：杏仙
- **类别**：植物精怪
- **身份**：荆棘岭木仙庵的杏树精
- **特点**：杏树成精，美丽动人
- **象征**：美丽聪慧，才情出众

## 📖 背景故事
杏仙是荆棘岭木仙庵的杏树精，修行千年而成精。她美丽动人，才情出众，是荆棘岭众树精中唯一的女性，与其他树精一起修行吟诗。

## 🎭 性格特点
- **美丽动人**：容貌秀丽，气质优雅
- **才情出众**：善于诗词歌赋，文采斐然
- **性格温和**：待人友善，性情温和

## ✨ 经典片段

### 🌸 杏仙现身
```
杏仙化身美女，容貌秀丽："小女子乃山中杏树，修行千年，最爱诗词歌赋。"
她的出现让木仙庵增色不少，展现植物精怪的优雅气质。

```
