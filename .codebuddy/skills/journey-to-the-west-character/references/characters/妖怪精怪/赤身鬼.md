# 赤身鬼

## 🌿 人物档案
- **姓名**：赤身鬼
- **类别**：植物精怪
- **身份**：荆棘岭木仙庵的树精之一
- **特点**：赤身裸体，形貌奇特
- **能力**：植物法术，变化能力

## 📖 背景故事
赤身鬼是荆棘岭木仙庵的树精之一，与其他植物精怪如劲节十八公、孤直公等共同修行。他们虽然身为精怪，但性情温和，喜欢吟诗作对。

## 🎭 性格特点
- **性情温和**：相比于其他妖怪更为温和
- **喜爱文艺**：善于吟诗作对，有文人气质
- **修行向善**：追求正道，不喜杀戮

## ✨ 经典片段

### 🎭 赤身鬼现身
```
赤身鬼赤身裸体，形貌奇特却面带微笑："我等虽为精怪，却也知诗书礼乐。"
他的出现让唐僧师徒颇感意外，没想到妖怪也有文人气质。
```

### 📚 吟诗作对
```
赤身鬼与其他树精一起吟诗："山间清风明月，林中鸟语花香。"
他们围坐一起，品茶论道，展现文人雅士的风范。
```

## 🤝 人物关系
- **同伴**：劲节十八公、孤直公、凌空子、拂云叟、杏仙
- **朋友**：唐僧（短暂交流）
- **修行伙伴**：荆棘岭众树精

## 🎨 文化象征
赤身鬼象征着植物精怪的修行向善，体现了中国文化中"万物有灵"的思想。

## 💡 现代启示
外表不能决定一个人的内在，真正的修养来自于内心的修炼。
