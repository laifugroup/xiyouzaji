# 灞波儿奔

## 📋 人物档案
- **姓名**：灞波儿奔
- **身份**：碧波潭黑鱼精，奔波儿灞的搭档
- **登场回目**：第六十二回《涤垢洗心惟扫塔 缚魔归正乃修身》

## 📖 背景故事
灞波儿奔是碧波潭万圣龙王麾下的黑鱼精，与鲇鱼精奔波儿灞是搭档。两人奉命在祭赛国金光寺看守偷来的佛宝舍利子。

当孙悟空扫塔寻宝时，发现了这两个小妖。灞波儿奔性格较为胆小，在被孙悟空抓住后，如实交代了盗宝的经过。

## 🎭 经典片段

### 1. **塔顶值班**
灞波儿奔与奔波儿灞在塔顶值班：

```
灞波儿奔："奔波儿灞，你说这佛宝真的那么神奇吗？"
奔波儿灞："当然！不然万圣龙王怎么会冒险来偷？"
灞波儿奔："可是我觉得心里不安，毕竟是佛门宝物..."
```

### 2. **被发现**
孙悟空扫塔时发现两个小妖：

```
孙悟空："哈哈！两个小妖怪在这里做什么？"
灞波儿奔："啊！是孙悟空！快跑！"
奔波儿灞："跑什么跑？跟他拼了！"
```

### 3. **被擒交代**
灞波儿奔被孙悟空抓住后交代实情：

```
孙悟空："说！佛宝是谁偷的？"
灞波儿奔："大圣饶命！是万圣龙王和九头虫偷的！"
孙悟空："详细说来！若有半句假话，小心你的脑袋！"
```

### 4. **详细供述**
灞波儿奔详细交代盗宝经过：

```
灞波儿奔："三年前，万圣龙王和驸马九头虫变化而来..."
孙悟空："他们现在在哪里？"
灞波儿奔："在碧波潭龙宫，我们只是奉命看守的小妖..."
```

### 5. **带路寻宝**
灞波儿奔带孙悟空去找佛宝：

```
孙悟空："带我去碧波潭，若敢耍花样，小心你的性命！"
灞波儿奔："不敢不敢！小人这就带路！"
```

## 💡 性格特点

### 优点
- **老实交代**：被抓后如实供述
- **配合态度**：愿意带路帮助寻宝
- **知错能改**：认识到错误后配合改正

### 缺点
- **胆小怕事**：遇到强敌容易害怕
- **缺乏主见**：只是奉命行事的小妖
- **实力弱小**：法力低微，不堪一击

## 🔗 人物关系
- **奔波儿灞**：搭档，同为看守小妖
- **万圣龙王**：上司，盗宝的主谋
- **九头虫**：盗宝的同伙
- **孙悟空**：抓住他的对手

## 🌟 文化象征
灞波儿奔代表了"小人物"的形象：
- **底层执行者**：只是奉命行事的小角色
- **无奈从命**：在强权下不得不服从
- **知错能改**：认识到错误后愿意改正

## 💭 现代启示
1. **明辨是非**：要能分辨什么该做，什么不该做
2. **勇于认错**：犯错后要勇于承认和改正
3. **配合调查**：在正确的事情上要积极配合

灞波儿奔的故事虽然简单，但反映了小人物在大事件中的无奈和选择。
