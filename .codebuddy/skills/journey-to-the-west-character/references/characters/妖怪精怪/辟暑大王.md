# 辟暑大王

## 🐾 人物档案
- **姓名**：辟暑大王
- **类别**：妖怪精怪
- **身份**：金平府犀牛精三兄弟之一
- **特点**：能够辟暑，不畏炎热
- **能力**：精通炎热法术

## 📖 背景故事
辟暑大王是金平府的三只犀牛精之一，与辟寒大王、辟尘大王是兄弟。他们假扮佛像骗取香油，被孙悟空识破后展开激战。

## 🎭 性格特点
- **炎热法术**：掌握炎热相关的法术
- **配合默契**：与两个兄弟战术配合
- **贪婪狡诈**：善于利用百姓的信仰

## ✨ 经典片段

### 🔥 炎热法术
```
辟暑大王施展法术，热浪滚滚："看我炎热之术！"
他口中喷出火焰，周围温度骤升，展现强大的炎热法术。
```

### 🤝 兄弟配合
```
辟暑大王与两个兄弟配合："大哥用寒，我用热，让这猴子难以应对！"
他们兄弟三人的法术配合让孙悟空也感到棘手。
```

## 🤝 人物关系
- **兄弟**：辟寒大王、辟尘大王
- **对手**：孙悟空、四木禽星
- **受害者**：金平府百姓

## 🎨 文化象征
辟暑大王象征着炎热与燥动，体现了自然力量的强大。

## 💡 现代启示
贪婪终将招致惩罚，利用他人信仰谋利是不可取的。
