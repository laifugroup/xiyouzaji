# 凌空子

## 🌿 人物档案
- **姓名**：凌空子
- **类别**：植物精怪
- **身份**：荆棘岭木仙庵的桧树精
- **特点**：桧树成精，气质飘逸
- **象征**：清高飘逸，超凡脱俗

## 📖 背景故事
凌空子是荆棘岭木仙庵的桧树精，修行千年而成精。他气质飘逸，性格清高，与其他树精一起在木仙庵修行，喜欢吟诗作对。

## 🎭 性格特点
- **气质飘逸**：具有桧树的清高气质
- **性格清高**：为人清高，不随俗流
- **学识渊博**：修行千年，知识丰富

## ✨ 经典片段

### 🌿 桧树显形
```
凌空子化身文人，气质飘逸："贫道乃山中桧树，修行千年，最爱清静。"
他的形象清雅脱俗，展现文人雅士的风范。
```

### 📚 清高气质
```
凌空子轻摇羽扇："世间纷扰，不如山中清静。修行之人，当以清心为上。"
他的话语中透露出清高脱俗的气质。
```

## 🤝 人物关系
- **同伴**：劲节十八公、孤直公、拂云叟、杏仙、赤身鬼
- **朋友**：唐僧（学问交流）

## 🎨 文化象征
凌空子象征着桧树的清高气质，体现了中国文化中对文人雅士的推崇。

## 💡 现代启示
清心寡欲，保持内心的宁静，才能在纷扰的世间保持清醒。
