# 雪狮

## 🐾 人物档案
- **姓名**：雪狮
- **类别**：妖怪精怪
- **身份**：狮驼岭三大王之一，黄狮精的兄弟
- **武器**：刀、枪等兵器
- **特点**：白色毛发，凶猛异常

## 📖 背景故事
雪狮是狮驼岭三大王之一，与黄狮精、猱狮等兄弟共同占据狮驼岭。他们统领众多小妖，威震一方，是取经路上的重要阻碍之一。

## 🎭 性格特点
- **凶猛好斗**：战斗力强，勇猛异常
- **兄弟情深**：与其他狮精兄弟团结一致
- **狂妄自大**：自恃力量强大，不把对手放在眼里

## ✨ 经典片段

### 🦁 雪狮发威
```
雪狮白毛如雪，威风凛凛站在山头上，大声喝道："哪里来的和尚，敢闯我狮驼岭！"
他的声音如同惊雷，震得山石滚动，小妖们纷纷跪拜。
```

### 🤺 激战孙悟空
```
雪狮挥舞大刀，与孙悟空战在一处："你这猴子，有什么本事，敢与我兄弟为敌！"
他刀法凌厉，每一刀都带着寒光，与孙悟空斗得难解难分。
```

## 🤝 人物关系
- **兄弟**：黄狮精、猱狮、狻猊等
- **对手**：孙悟空、猪八戒、沙僧
- **部下**：狮驼岭众小妖

## 🎨 文化象征
雪狮象征着凶猛与力量，体现了动物精怪在神话中的强大形象。

## 💡 现代启示
兄弟团结力量大，但过于自负容易招致失败。
