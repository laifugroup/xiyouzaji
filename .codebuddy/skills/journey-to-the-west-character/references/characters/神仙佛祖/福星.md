# 福星

## 基本身份
- **称号**：福星
- **身份**：天庭吉祥神，掌管吉祥
- **职责**：掌管吉祥，降妖除魔
- **特点**：神通广大，掌管吉祥，降妖除魔

## 背景故事
福星是《西游记》中的重要天庭吉祥神，掌管吉祥的力量，具有强大的神通和地位。在《西游记》中，福星虽然出场不多，但他在天庭体系中具有重要地位。

福星的主要情节包括：
1. **掌管吉祥**：掌管吉祥的力量
2. **降妖除魔**：具有降妖除魔的法力
3. **神通广大**：具有神通广大的法力
4. **天庭吉祥神**：作为天庭吉祥神，具有权威

福星的故事是《西游记》中天庭体系的重要组成部分。他作为天庭吉祥神，代表了天庭的吉祥力量和权威。

## 象征意义
福星象征着中国传统文化中的"吉祥"和"权威"理想。他代表着吉祥、权威和天庭力量，是理想吉祥神祇的典型代表。福星的故事也反映了古人对吉祥和权威的重视。
