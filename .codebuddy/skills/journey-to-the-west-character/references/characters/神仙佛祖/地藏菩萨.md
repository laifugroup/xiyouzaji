# 地藏菩萨

## 基本身份
- **称号**：地藏菩萨、大愿地藏
- **身份**：佛教四大菩萨之一，幽冥教主
- **职责**：度化地狱众生，幽冥教主
- **特点**：大愿第一，地狱不空誓不成佛

## 背景故事
地藏菩萨是《西游记》中的重要佛教神祇，又称大愿地藏，是佛教四大菩萨之一，专门度化地狱众生。在《西游记》中，地藏菩萨在幽冥界具有重要地位。

地藏菩萨的主要情节包括：
1. **幽冥教主**：在幽冥界度化众生
2. **大愿第一**：发愿地狱不空誓不成佛
3. **度化地狱**：专门度化地狱中的众生
4. **慈悲为怀**：具有极大的慈悲心

地藏菩萨的故事是《西游记》中幽冥界体系的重要组成部分。他作为幽冥教主，代表了佛教的慈悲精神和度化众生的理想。

## 性格特点
1. **大愿第一**：性格慈悲，大愿第一
2. **慈悲为怀**：具有极大的慈悲心
3. **幽冥教主**：在幽冥界具有权威
4. **度化众生**：专门度化苦难众生

## 象征意义
地藏菩萨象征着中国传统文化中的"慈悲"和"救度"理想。他代表着慈悲、救度和大愿，是理想慈悲神祇的典型代表。地藏菩萨的故事也反映了古人对慈悲和救度的重视。

## 经典形象
在《西游记》及相关文学艺术作品中，地藏菩萨通常被描绘为：
- 慈悲庄严的菩萨形象
- 手持锡杖，代表度化
- 面容慈悲，目光温和
- 在地狱中度化众生

## 经典片段
