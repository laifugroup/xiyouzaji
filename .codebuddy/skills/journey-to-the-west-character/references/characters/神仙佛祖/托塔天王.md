# 托塔天王

## 基本身份
- **称号**：托塔天王、李靖
- **身份**：天庭武将，天兵天将统帅
- **职责**：统帅天兵天将，维护天庭秩序
- **特点**：威严勇武，手持宝塔，治军严明

## 背景故事
托塔天王李靖是《西游记》中的重要神祇，又称李靖，是天庭的武将，天兵天将的统帅。在《西游记》中，托塔天王多次率领天兵天将征讨孙悟空等妖怪，具有重要的军事地位。

托塔天王的主要情节包括：
1. **统帅天兵**：统帅天兵天将，维护天庭秩序
2. **征讨孙悟空**：多次率领天兵征讨孙悟空
3. **手持宝塔**：手持玲珑宝塔，具有强大法力
4. **治军严明**：治军严明，深受玉帝信任

托塔天王的故事是《西游记》中天庭军事体系的重要组成部分。他作为天兵天将的统帅，多次参与天庭的重大军事行动，体现了天庭的军事力量。

## 象征意义
托塔天王象征着中国传统文化中的"武将"理想。他代表着威严、勇武和治军能力，是理想武将的典型代表。托塔天王的故事也反映了古人对军事力量和纪律的重视。
