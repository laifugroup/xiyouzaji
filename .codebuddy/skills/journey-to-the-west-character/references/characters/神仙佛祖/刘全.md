# 刘全

## 基本身份
- **称号**：刘全、进瓜者
- **身份**：普通凡人，李翠莲的丈夫
- **职责**：务农为生
- **特点**：忠厚老实，重情重义

## 背景故事
刘全是《西游记》中的凡人角色，他是一个普通的农民，李翠莲的丈夫。在《西游记》中，刘全虽然出场不多，但他的故事体现了因果报应和夫妻情深的主题。

刘全的主要情节包括：
1. **妻子自杀**：李翠莲因将金钗施舍给僧人而被刘全误会，愤而自杀
2. **还阳献瓜**：刘全为赎罪，自愿赴阴司献瓜，助妻子还阳
3. **夫妻团圆**：在地府受审后，夫妻得以还阳团圆

刘全的故事是《西游记》中体现"善有善报"观念的重要情节。他因误会导致妻子自杀，后为赎罪自愿赴阴司，最终夫妻得以团圆，体现了因果报应和夫妻情深。

## 性格特点
1. **忠厚老实**：为人朴实，不善言辞
2. **重情重义**：对妻子情深意重
3. **勇于担当**：为赎罪自愿赴阴司
4. **知错能改**：认识到自己的错误后积极改正

## 象征意义
刘全象征着中国传统文化中的"忠厚"品质。他代表着普通人的善良和重情重义，是"善有善报"观念的具体体现。刘全的故事也反映了古人对夫妻关系和家庭伦理的重视。
