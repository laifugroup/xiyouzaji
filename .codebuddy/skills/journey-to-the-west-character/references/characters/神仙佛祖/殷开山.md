# 殷开山

## 基本身份
- **称号**：殷开山、殷丞相
- **身份**：唐朝丞相，殷温娇之父
- **职责**：辅佐皇帝，治理国家
- **特点**：位高权重，正直无私，关爱家人

## 背景故事
殷开山是《西游记》中的重要凡人角色，他是唐朝的丞相，殷温娇的父亲，唐僧的外祖父。在《西游记》中，殷开山虽然出场不多，但他在唐僧身世故事中扮演了重要角色。

殷开山的主要情节包括：
1. **位高权重**：作为唐朝丞相，地位显赫
2. **女儿婚事**：为女儿殷温娇举办抛绣球招亲
3. **外孙寻亲**：多年后与外孙唐僧相认
4. **主持正义**：为女儿女婿沉冤昭雪

殷开山的故事是唐僧身世的重要组成部分。他作为朝廷重臣，本应享受荣华富贵，却因女儿女婿的遭遇而经历家庭悲剧，最终为亲人主持正义，体现了家族亲情的重要性。

## 性格特点
1. **位高权重**：作为丞相，具有很高的政治地位
2. **正直无私**：为官清廉，主持正义
3. **关爱家人**：对女儿和外孙充满关爱
4. **重情重义**：重视家族亲情和道义

## 象征意义
殷开山象征着中国传统文化中的"重臣"理想。他代表着权力、正直和家族责任，是儒家政治理想中理想大臣的体现。殷开山的故事也反映了古代士大夫对家族荣誉和道义的重视。

## 经典形象
在《西游记》及相关文学艺术作品中，殷开山通常被描绘为：
- 威严庄重的丞相形象
- 身穿朝服，头戴官帽
- 面容严肃，目光深邃
- 在朝堂上处理政务
