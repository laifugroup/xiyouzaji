# 弥勒佛

## 基本身份
- **称号**：弥勒佛、弥勒菩萨、未来佛
- **身份**：佛教未来佛，释迦牟尼佛的继任者
- **职责**：未来成佛，普度众生
- **特点**：笑口常开，大肚能容，未来佛

## 背景故事
弥勒佛是《西游记》中的重要佛教神祇，又称未来佛，是释迦牟尼佛的继任者，将在未来成佛普度众生。在《西游记》中，弥勒佛虽然出场不多，但他在佛教体系中具有重要地位。

弥勒佛的主要情节包括：
1. **未来佛**：作为未来佛，将在未来成佛
2. **笑口常开**：形象笑口常开，大肚能容
3. **普度众生**：未来将普度众生
4. **佛教地位**：在佛教中具有重要地位

弥勒佛的故事是《西游记》中佛教体系的重要组成部分。他作为未来佛，代表了佛教的未来希望和普度众生的理想。
