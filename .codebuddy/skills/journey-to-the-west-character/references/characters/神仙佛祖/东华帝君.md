# 东华帝君

## 基本身份
- **称号**：东华帝君
- **身份**：天庭东方神，掌管东方
- **职责**：掌管东方，降妖除魔
- **特点**：神通广大，掌管东方，降妖除魔

## 背景故事
东华帝君是《西游记》中的重要天庭东方神，掌管东方的力量，具有强大的神通和地位。在《西游记》中，东华帝君虽然出场不多，但他在天庭体系中具有重要地位。

东华帝君的主要情节包括：
1. **掌管东方**：掌管东方的力量
2. **降妖除魔**：具有降妖除魔的法力
3. **神通广大**：具有神通广大的法力
4. **天庭东方神**：作为天庭东方神，具有权威

东华帝君的故事是《西游记》中天庭体系的重要组成部分。他作为天庭东方神，代表了天庭的东方力量和权威。

## 象征意义
东华帝君象征着中国传统文化中的"东方"和"权威"理想。他代表着东方、权威和天庭力量，是理想东方神祇的典型代表。东华帝君的故事也反映了古人对东方和权威的重视。
