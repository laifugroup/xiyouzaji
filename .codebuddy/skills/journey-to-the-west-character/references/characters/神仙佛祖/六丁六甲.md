# 六丁六甲

## 基本身份
- **称号**：六丁六甲
- **身份**：天庭护法神，掌管护法
- **职责**：护法护持，降妖除魔
- **特点**：神通广大，护法护持，降妖除魔

## 背景故事
六丁六甲是《西游记》中的重要天庭护法神，掌管护法的力量，具有强大的神通和地位。在《西游记》中，六丁六甲虽然出场不多，但他们在天庭体系中具有重要地位。

六丁六甲的主要情节包括：
1. **护法护持**：护法护持的力量
2. **降妖除魔**：具有降妖除魔的法力
3. **神通广大**：具有神通广大的法力
4. **天庭护法神**：作为天庭护法神，具有权威

六丁六甲的故事是《西游记》中天庭体系的重要组成部分。他们作为天庭护法神，代表了天庭的护法力量和权威。

六丁六甲象征着中国传统文化中的"护法"和"权威"理想。他们代表着护法、权威和天庭力量，是理想护法神祇的典型代表。六丁六甲的故事也反映了古人对护法和权威的重视。
