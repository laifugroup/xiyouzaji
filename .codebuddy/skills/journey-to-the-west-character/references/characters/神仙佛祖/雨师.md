# 雨师

## 基本身份
- **称号**：雨师、雨神
- **身份**：天庭雨神，掌管雨
- **职责**：掌管雨，降妖除魔
- **特点**：神通广大，掌管雨，降妖除魔

## 背景故事
雨师是《西游记》中的重要天庭雨神，掌管雨的力量，具有强大的神通和地位。在《西游记》中，雨师虽然出场不多，但他在天庭体系中具有重要地位。

雨师的主要情节包括：
1. **掌管雨**：掌管雨的力量
2. **降妖除魔**：具有降妖除魔的法力
3. **神通广大**：具有神通广大的法力
4. **天庭雨神**：作为天庭雨神，具有权威

雨师的故事是《西游记》中天庭体系的重要组成部分。他作为天庭雨神，代表了天庭的雨力量和权威。

## 象征意义
雨师象征着中国传统文化中的"雨"和"权威"理想。他代表着雨、权威和天庭力量，是理想雨神神祇的典型代表。雨师的故事也反映了古人对雨和权威的重视。
