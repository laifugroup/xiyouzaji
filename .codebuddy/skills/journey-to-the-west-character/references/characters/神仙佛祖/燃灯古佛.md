# 燃灯古佛

## 基本身份
- **称号**：燃灯古佛、燃灯佛
- **身份**：佛教过去佛，释迦牟尼佛的前任
- **职责**：过去成佛，普度众生
- **特点**：智慧深邃，过去佛，佛教始祖

## 背景故事
燃灯古佛是《西游记》中的重要佛教神祇，又称过去佛，是释迦牟尼佛的前任，在过去成佛普度众生。在《西游记》中，燃灯古佛虽然出场不多，但他在佛教体系中具有重要地位。

燃灯古佛的主要情节包括：
1. **过去佛**：作为过去佛，在过去成佛
2. **智慧深邃**：具有深邃的智慧
3. **佛教始祖**：在佛教中具有始祖地位
4. **与释迦牟尼**：是释迦牟尼佛的前任

燃灯古佛的故事是《西游记》中佛教体系的重要组成部分。他作为过去佛，代表了佛教的历史渊源和智慧传承。

## 性格特点
1. **智慧深邃**：性格智慧，深邃难测
2. **慈悲为怀**：具有慈悲心，普度众生
3. **过去佛**：代表佛教的历史渊源
4. **佛教始祖**：具有始祖的威严

## 象征意义
燃灯古佛象征着中国传统文化中的"智慧"和"传承"理想。他代表着智慧、传承和历史渊源，是理想佛教神祇的典型代表。燃灯古佛的故事也反映了古人对智慧和传承的重视。

## 经典形象

- 智慧深邃的老者形象

- 代表智慧和传承
