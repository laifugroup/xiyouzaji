# 文殊菩萨

## 基本身份
- **称号**：文殊菩萨、文殊师利菩萨、大智文殊
- **身份**：佛教四大菩萨之一，智慧菩萨
- **职责**：代表智慧，普度众生
- **特点**：智慧第一，手持智慧剑，骑乘青狮

## 背景故事
文殊菩萨是《西游记》中的重要佛教神祇，又称大智文殊，是佛教四大菩萨之一，代表智慧。在《西游记》中，文殊菩萨虽然出场不多，但他在佛教体系中具有重要地位。

文殊菩萨的主要情节包括：
1. **智慧第一**：在佛教中智慧第一
2. **手持智慧剑**：手持智慧剑斩断烦恼
3. **骑乘青狮**：骑乘青狮象征智慧威猛
4. **普度众生**：以智慧普度众生

文殊菩萨的故事是《西游记》中佛教体系的重要组成部分。他作为智慧菩萨，代表了佛教的智慧精神和普度众生的理想。

## 性格特点
1. **智慧深邃**：性格智慧，深邃难测
2. **慈悲为怀**：具有慈悲心，普度众生
3. **智慧第一**：在佛教中智慧第一
4. **威严庄重**：具有菩萨的威严

## 象征意义
文殊菩萨象征着中国传统文化中的"智慧"理想。他代表着智慧、学识和理性，是理想智慧神祇的典型代表。文殊菩萨的故事也反映了古人对智慧和学识的重视。
