# 四大天王

## 基本身份
- **称号**：四大天王、护法天王
- **身份**：天庭护法，佛教护法
- **职责**：护持天庭，护持佛法
- **特点**：各有所长，护法有力，护持佛法

## 背景故事
四大天王是《西游记》中的重要天庭护法，又称护法天王，是天庭和佛教的重要护法。在《西游记》中，四大天王具有强大的护法能力和地位。

四大天王的主要成员包括：
1. **增长天王**：手持宝剑，护持东方
2. **持国天王**：手持琵琶，护持南方
3. **多闻天王**：手持宝伞，护持西方
4. **广目天王**：手持蛇龙，护持北方

四大天王的故事是《西游记》中天庭体系的重要组成部分。他们作为天庭护法，代表了天庭的护法精神和秩序维护。

## 性格特点
1. **护法有力**：性格威严，护法有力
2. **各有所长**：四大天王各有所长
3. **护持佛法**：护持佛法的传播
4. **天庭护法**：作为天庭护法，具有权威

## 象征意义
四大天王象征着中国传统文化中的"护法"和"秩序"理想。他们代表着护法、秩序和天庭权威，是理想护法神祇的典型代表。四大天王的故事也反映了古人对护法和秩序的重视。

## 经典形象
在《西游记》及相关文学艺术作品中，四大天王通常被描绘为：
- 威严庄重的护法形象
- 各持法器，护持四方
- 面容威严，目光锐利
- 天庭护法的英姿

## 经典片段
1. **护法四方**：护持四方的场景
2. **各有所长**：四大天王各显神通
