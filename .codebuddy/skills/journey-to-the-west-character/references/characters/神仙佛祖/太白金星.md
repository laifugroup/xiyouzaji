# 太白金星

## 基本身份
- **称号**：太白金星、李长庚
- **身份**：天庭文臣，玉帝使者
- **职责**：传达玉帝旨意，调解天庭事务
- **特点**：温和睿智，善于调解，天庭使者

## 背景故事
太白金星是《西游记》中的重要神祇，又称李长庚，是天庭的文臣，玉皇大帝的重要使者。在《西游记》中，太白金星多次作为玉帝的使者出现，具有重要的调解作用。

太白金星的主要情节包括：
1. **招安孙悟空**：奉玉帝之命招安孙悟空
2. **调解天庭事务**：多次调解天庭与孙悟空的关系
3. **传达玉帝旨意**：作为玉帝的使者传达旨意
4. **温和睿智**：以温和睿智的方式处理事务

太白金星的故事是《西游记》中天庭体系的重要组成部分。他作为玉帝的使者，多次调解天庭与孙悟空的关系，体现了温和睿智的处事风格。

## 性格特点
1. **温和睿智**：性格温和，处事睿智
2. **善于调解**：擅长调解各种矛盾
3. **忠诚可靠**：对玉帝忠诚可靠
4. **处事圆融**：处事圆融，善于沟通

## 象征意义
太白金星象征着中国传统文化中的"文臣"理想。他代表着温和、睿智和调解能力，是理想文臣的典型代表。太白金星的故事也反映了古人对温和处事和智慧调解的推崇。
