# 木叉

## 基本身份
- **称号**：木叉、惠岸行者
- **身份**：托塔天王次子，观音菩萨护法
- **职责**：协助观音菩萨，护法修行
- **特点**：忠诚护法，修行精进，观音弟子

## 背景故事
木叉是《西游记》中的重要神祇，又称惠岸行者，是托塔天王李靖的次子，观音菩萨的护法弟子。在《西游记》中，木叉作为观音菩萨的弟子，具有重要的护法作用。

木叉的主要情节包括：
1. **观音护法**：作为观音菩萨的护法弟子
2. **修行精进**：跟随观音菩萨修行
3. **协助观音**：多次协助观音菩萨
4. **与孙悟空**：曾与孙悟空有过交集

木叉的故事是《西游记》中观音菩萨体系的重要组成部分。他作为观音菩萨的护法弟子，体现了佛教护法弟子的修行精神。

## 性格特点
1. **忠诚护法**：对观音菩萨忠诚，护法精进
2. **修行精进**：修行精进，道心坚定
3. **温和谦逊**：性格温和，谦逊有礼
4. **观音弟子**：作为观音弟子，具有慈悲心

## 象征意义
木叉象征着中国传统文化中的"护法"理想。他代表着忠诚、精进和护法精神，是理想护法弟子的典型代表。木叉的故事也反映了古人对佛教护法精神的推崇。

## 经典形象

- 跟随观音菩萨修行
