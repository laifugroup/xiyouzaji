# 接引佛祖

## 基本身份
- **称号**：接引佛祖、撑无底船者
- **身份**：佛教接引佛，渡人过河
- **职责**：接引众生渡过苦海，到达彼岸
- **特点**：撑无底船，接引众生，渡人过河

## 背景故事
接引佛祖是《西游记》中的重要佛教神祇，又称撑无底船者，是专门接引众生渡过苦海到达彼岸的佛。在《西游记》中，接引佛祖在唐僧取经的最后阶段出现，具有重要的象征意义。

接引佛祖的主要情节包括：
1. **撑无底船**：撑无底船接引众生
2. **渡人过河**：帮助众生渡过苦海
3. **到达彼岸**：接引众生到达彼岸
4. **象征意义**：代表佛教的渡人理想

接引佛祖的故事是《西游记》中佛教体系的重要组成部分。他作为接引佛，代表了佛教的渡人精神和到达彼岸的理想。

## 性格特点
1. **慈悲为怀**：性格慈悲，接引众生
2. **渡人过河**：专门渡人过河到达彼岸
3. **象征意义**：具有重要的象征意义
4. **接引众生**：接引众生脱离苦海

## 象征意义
接引佛祖象征着中国传统文化中的"渡人"和"解脱"理想。他代表着渡人、接引和解脱，是理想渡人神祇的典型代表。接引佛祖的故事也反映了古人对解脱和彼岸的向往。

## 经典形象
在《西游记》及相关文学艺术作品中，接引佛祖通常被描绘为：
- 慈悲庄严的佛祖形象
- 撑无底船，接引众生
- 面容慈悲，目光温和
- 在河边渡人过河
