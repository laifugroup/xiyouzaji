# 高翠兰

## 基本身份
- **称号**：高翠兰、高小姐
- **身份**：高老庄小姐，高太公之女
- **职责**：闺中待嫁，相夫教子
- **特点**：美丽贤淑，命运多舛，坚贞不屈

## 背景故事
高翠兰是《西游记》中的重要凡人角色，她是高老庄的小姐，高太公的女儿。在《西游记》中，高翠兰虽然出场不多，但她在猪八戒的故事中扮演了关键角色。

高翠兰的主要情节包括：
1. **招婿入赘**：被父亲招猪八戒为婿
2. **发现真相**：发现丈夫猪八戒是妖怪
3. **被困受苦**：被猪八戒困在后院受苦
4. **得救团圆**：被唐僧师徒所救，重获自由

高翠兰的故事是猪八戒加入取经团队的重要契机。她作为富家小姐，本应享受幸福生活，却因嫁给妖怪而命运多舛，最终在唐僧师徒的帮助下重获自由。

## 性格特点
1. **美丽贤淑**：出身富家，品貌端庄
2. **命运多舛**：人生遭遇重大变故
3. **坚贞不屈**：在困境中保持坚贞
4. **善良温柔**：心地善良，性情温柔

## 象征意义
高翠兰象征着中国传统文化中的"闺秀"理想。她代表着美丽、贤淑和坚贞的品质，是儒家女性理想的体现。高翠兰的命运也反映了古代女性在婚姻选择上的无奈和坚韧。

## 经典形象
在《西游记》及相关文学艺术作品中，高翠兰通常被描绘为：
- 美丽端庄的闺秀形象
- 身穿华服，面容姣好
- 神情忧郁，目光坚定
- 被困后院的可怜形象
