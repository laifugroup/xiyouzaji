# 药师佛

## 基本身份
- **称号**：药师佛、药师琉璃光如来
- **身份**：佛教东方净琉璃世界教主
- **职责**：医治众生疾苦，消灾延寿
- **特点**：医治众生，消灾延寿，东方教主

## 背景故事
药师佛是《西游记》中的重要佛教神祇，又称药师琉璃光如来，是佛教东方净琉璃世界的教主，专门医治众生疾苦。在《西游记》中，药师佛虽然出场不多，但他在佛教体系中具有重要地位。

药师佛的主要情节包括：
1. **东方教主**：是东方净琉璃世界的教主
2. **医治众生**：专门医治众生的疾苦
3. **消灾延寿**：具有消灾延寿的法力
4. **慈悲为怀**：具有极大的慈悲心

药师佛的故事是《西游记》中佛教体系的重要组成部分。他作为医治众生的佛，代表了佛教的慈悲精神和医治众生的理想。

## 性格特点
1. **慈悲为怀**：性格慈悲，医治众生
2. **消灾延寿**：具有消灾延寿的法力
3. **东方教主**：在东方世界具有权威
4. **医治疾苦**：专门医治众生的疾苦

## 象征意义
药师佛象征着中国传统文化中的"医治"和"慈悲"理想。他代表着医治、消灾和慈悲，是理想医治神祇的典型代表。药师佛的故事也反映了古人对健康和长寿的重视。

## 经典形象
在《西游记》及相关文学艺术作品中，药师佛通常被描绘为：
- 慈悲庄严的佛祖形象
- 手持药壶或药草，代表医治
- 面容慈悲，目光温和
- 在东方净琉璃世界教化众生
