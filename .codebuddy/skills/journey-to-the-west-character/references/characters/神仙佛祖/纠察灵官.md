# 纠察灵官

## 基本身份
- **称号**：纠察灵官
- **身份**：天庭监察神，掌管监察
- **职责**：掌管监察，降妖除魔
- **特点**：神通广大，掌管监察，降妖除魔

## 背景故事
纠察灵官是《西游记》中的重要天庭监察神，掌管监察的力量，具有强大的神通和地位。在《西游记》中，纠察灵官虽然出场不多，但他在天庭体系中具有重要地位。

纠察灵官的主要情节包括：
1. **掌管监察**：掌管监察的力量
2. **降妖除魔**：具有降妖除魔的法力
3. **神通广大**：具有神通广大的法力
4. **天庭监察神**：作为天庭监察神，具有权威

纠察灵官的故事是《西游记》中天庭体系的重要组成部分。他作为天庭监察神，代表了天庭的监察力量和权威。

## 性格特点

## 象征意义
纠察灵官象征着中国传统文化中的"监察"和"权威"理想。他代表着监察、权威和天庭力量，是理想监察神祇的典型代表。纠察灵官的故事也反映了古人对监察和权威的重视。
