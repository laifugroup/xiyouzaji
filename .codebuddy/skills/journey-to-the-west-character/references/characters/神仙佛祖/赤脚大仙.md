# 赤脚大仙

## 基本身份
- **称号**：赤脚大仙
- **身份**：天庭散仙，逍遥神仙
- **职责**：云游四方，逍遥自在
- **特点**：赤脚行走，逍遥自在，不拘小节

## 背景故事
赤脚大仙是《西游记》中的天庭散仙，以赤脚行走、逍遥自在而闻名。在《西游记》中，赤脚大仙虽然出场不多，但他代表了天庭中逍遥自在的一类神仙。

赤脚大仙的主要情节包括：
1. **赤脚行走**：常年赤脚行走，不拘小节
2. **逍遥自在**：云游四方，逍遥自在
3. **天庭散仙**：作为天庭散仙，不受拘束
4. **与孙悟空**：曾与孙悟空有过交集

赤脚大仙的故事是《西游记》中天庭体系的重要组成部分。他代表了神仙中逍遥自在、不拘小节的一类，体现了神仙世界的多样性。

## 性格特点
1. **逍遥自在**：性格逍遥，不受拘束
2. **不拘小节**：生活简朴，不拘小节
3. **赤脚行走**：常年赤脚，体现自然
4. **随和友善**：性格随和，待人友善

## 象征意义
赤脚大仙象征着中国传统文化中的"逍遥"理想。他代表着自由、自然和随性，是逍遥神仙的典型代表。赤脚大仙的故事也反映了古人对自由自在生活的向往。

## 经典形象
在《西游记》及相关文学艺术作品中，赤脚大仙通常被描绘为：
- 逍遥自在的老者形象
- 赤脚行走，衣着简朴
- 面容和善，目光清澈
- 手持拐杖或葫芦
