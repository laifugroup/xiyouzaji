# 普贤菩萨

## 基本身份
- **称号**：普贤菩萨、大行普贤
- **身份**：佛教四大菩萨之一，行愿菩萨
- **职责**：代表行愿，普度众生
- **特点**：行愿第一，骑乘白象，实践佛法

## 背景故事
普贤菩萨是《西游记》中的重要佛教神祇，又称大行普贤，是佛教四大菩萨之一，代表行愿和实践。在《西游记》中，普贤菩萨虽然出场不多，但他在佛教体系中具有重要地位。

普贤菩萨的主要情节包括：
1. **行愿第一**：在佛教中行愿第一
2. **骑乘白象**：骑乘白象象征行愿稳重
3. **实践佛法**：强调佛法的实践
4. **普度众生**：以行愿普度众生

普贤菩萨的故事是《西游记》中佛教体系的重要组成部分。他作为行愿菩萨，代表了佛教的实践精神和普度众生的理想。

## 性格特点
1. **行愿第一**：性格实践，行愿第一
2. **慈悲为怀**：具有慈悲心，普度众生
3. **稳重踏实**：性格稳重，踏实实践
4. **实践精神**：强调佛法的实践

## 象征意义
普贤菩萨象征着中国传统文化中的"实践"理想。他代表着实践、行动和行愿，是理想实践神祇的典型代表。普贤菩萨的故事也反映了古人对实践和行动的重视。
