# 二郎神

## 基本身份
- **称号**：二郎神、杨戬、显圣真君
- **身份**：天庭战神，玉帝外甥
- **职责**：降妖除魔，护持天庭
- **特点**：神通广大，八九玄功，三只天眼

## 背景故事
二郎神是《西游记》中的重要天庭战神，又称杨戬、显圣真君，是玉帝的外甥。在《西游记》中，二郎神具有强大的神通和地位，是天庭的重要武将。

二郎神的主要情节包括：
1. **八九玄功**：具有八九玄功的神通
2. **三只天眼**：拥有三只天眼，能看破一切
3. **降妖除魔**：具有降妖除魔的强大法力
4. **与孙悟空**：曾与孙悟空激烈战斗

二郎神的故事是《西游记》中天庭体系的重要组成部分。他作为天庭战神，代表了天庭的武力和权威。

## 性格特点
1. **神通广大**：性格高傲，神通广大
2. **降妖除魔**：专门降妖除魔护持天庭
3. **三只天眼**：具有三只天眼的特殊能力
4. **八九玄功**：掌握八九玄功的神通

## 象征意义
二郎神象征着中国传统文化中的"武力"和"权威"理想。他代表着武力、权威和天庭秩序，是理想战神的典型代表。二郎神的故事也反映了古人对武力和权威的重视。
