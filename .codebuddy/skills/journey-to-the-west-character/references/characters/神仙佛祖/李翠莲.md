# 李翠莲

## 基本身份
- **称号**：李翠莲、进瓜者之妻
- **身份**：普通凡人，刘全的妻子
- **职责**：相夫教子
- **特点**：善良贤惠，性情刚烈

## 背景故事
李翠莲是《西游记》中的凡人角色，她是刘全的妻子，一个普通的农家妇女。在《西游记》中，李翠莲虽然出场不多，但她的故事体现了善良和贞烈的品质。

李翠莲的主要情节包括：
1. **施舍金钗**：将金钗施舍给化缘的僧人
2. **被夫误会**：因施舍金钗被刘全误会
3. **愤而自杀**：因被误会而愤然自杀
4. **还阳团圆**：在地府受审后得以还阳，与丈夫团圆

李翠莲的故事是《西游记》中体现"善有善报"观念的重要情节。她因善良施舍而遭误会，最终因善行得以还阳，体现了因果报应和善良品质的价值。

李翠莲象征着中国传统文化中的"贤妻"理想。她代表着善良、贞烈和重情重义的品质，是"善有善报"观念的具体体现。李翠莲的故事也反映了古人对女性品德和家庭伦理的重视。
