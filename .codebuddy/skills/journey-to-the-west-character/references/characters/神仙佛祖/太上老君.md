# 太上老君

## 基本身份
- **称号**：太上老君、道德天尊、老子
- **身份**：三清之一，道教始祖，炼丹大师
- **职责**：掌管炼丹炉，炼制仙丹，传授道法
- **特点**：智慧深邃，炼丹高手，道教始祖

## 背景故事
太上老君是《西游记》中的重要神祇，又称道德天尊，是道教三清之一，被尊为道教始祖。在《西游记》中，太上老君在天庭体系中具有重要地位，是炼丹和道法的权威。

太上老君的主要情节包括：
1. **炼丹大师**：掌管炼丹炉，炼制各种仙丹
2. **道法传授**：传授道法，培养弟子
3. **八卦炉**：拥有八卦炉，能炼制各种法宝
4. **与孙悟空**：曾将孙悟空放入八卦炉中炼化

太上老君的故事是《西游记》中道教体系的重要组成部分。他作为道教始祖和炼丹大师，在天庭中具有极高的地位和威望。

## 性格特点
1. **智慧深邃**：具有深邃的智慧和道法修为
2. **炼丹高手**：精通炼丹术，能炼制各种仙丹
3. **道教始祖**：作为道教始祖，具有崇高地位
4. **慈悲为怀**：对众生慈悲为怀

## 象征意义
太上老君象征着中国传统文化中的"道教"理想。他代表着智慧、道法和炼丹术，是道教文化的最高代表。太上老君的故事也反映了古人对长生不老和道法修炼的追求。

## 经典形象
在《西游记》及相关文学艺术作品中，太上老君通常被描绘为：
- 仙风道骨的老者形象
- 身穿道袍，手持拂尘
- 面容慈祥，目光深邃
- 在炼丹炉前炼制仙丹

## 经典片段

2. **道法传授**：向弟子传授道法
3. **与孙悟空**：将孙悟空放入八卦炉炼化
