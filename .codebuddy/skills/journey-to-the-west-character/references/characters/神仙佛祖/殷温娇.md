# 殷温娇

## 基本身份
- **称号**：殷温娇、唐僧之母
- **身份**：丞相殷开山之女，陈光蕊之妻
- **职责**：相夫教子
- **特点**：贤淑美丽，命运坎坷，坚贞不屈

## 背景故事
殷温娇是《西游记》中的重要凡人角色，她是唐僧的母亲，丞相殷开山的女儿。在《西游记》中，殷温娇的故事虽然简短，但对唐僧的身世和成长具有重要意义。

殷温娇的主要情节包括：
1. **抛绣球招亲**：通过抛绣球选中陈光蕊为夫
2. **随夫赴任**：随陈光蕊赴江州任刺史
3. **遭遇变故**：丈夫被水贼刘洪所害，被迫改嫁
4. **生子弃子**：生下唐僧后为保其性命而弃之江中
5. **沉冤得雪**：多年后冤情得以昭雪

殷温娇的故事是唐僧身世的重要组成部分。她作为丞相之女，本应生活优渥，却因丈夫遇害而命运多舛，为保护儿子而做出艰难抉择，体现了母爱的伟大。

## 性格特点
1. **贤淑美丽**：出身名门，品貌端庄
2. **命运坎坷**：人生遭遇重大变故
3. **坚贞不屈**：在逆境中保持坚贞
4. **母爱伟大**：为保护儿子不惜牺牲自己

## 象征意义
殷温娇象征着中国传统文化中的"贤妻良母"理想。她代表着美丽、贤淑和坚贞的品质，是儒家女性理想的体现。殷温娇的命运也反映了古代女性在命运面前的无奈和坚韧。

## 经典形象
在《西游记》及相关文学艺术作品中，殷温娇通常被描绘为：
- 美丽端庄的贵妇形象
- 身穿华服，面容姣好
- 神情忧郁，目光坚定
