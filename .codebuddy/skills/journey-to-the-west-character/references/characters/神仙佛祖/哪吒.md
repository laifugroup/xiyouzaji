# 哪吒

## 基本身份
- **称号**：哪吒、三太子、中坛元帅
- **身份**：托塔天王之子，天庭武将
- **职责**：协助父亲统帅天兵，降妖除魔
- **特点**：少年英雄，法力高强，三头六臂

## 背景故事
哪吒是《西游记》中的重要神祇，又称三太子，是托塔天王李靖的儿子，天庭的武将。在《西游记》中，哪吒多次协助父亲征讨孙悟空等妖怪，具有重要的战斗能力。

哪吒的主要情节包括：
1. **少年英雄**：作为少年英雄，法力高强
2. **三头六臂**：具有三头六臂的神通
3. **协助父亲**：多次协助父亲托塔天王
4. **降妖除魔**：参与天庭的降妖除魔行动

哪吒的故事是《西游记》中天庭军事体系的重要组成部分。他作为少年英雄，具有强大的战斗能力，是天庭重要的战斗力量。

## 性格特点
1. **少年英雄**：性格勇敢，少年英雄气概
2. **法力高强**：具有强大的法力
3. **三头六臂**：具有三头六臂的神通
4. **忠诚勇敢**：对父亲忠诚，勇敢无畏

## 象征意义
哪吒象征着中国传统文化中的"少年英雄"理想。他代表着勇敢、忠诚和战斗能力，是少年英雄的典型代表。哪吒的故事也反映了古人对少年英雄的推崇。

## 经典形象
在《西游记》及相关文学艺术作品中，哪吒通常被描绘为：
- 英俊勇敢的少年形象
- 身穿铠甲，手持火尖枪
- 面容英俊，目光锐利
- 三头六臂的战斗姿态

## 经典片段
