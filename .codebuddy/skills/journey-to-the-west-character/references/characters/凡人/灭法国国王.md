# 灭法国国王

## 📋 人物档案
- **身份**：灭法国国王
- **登场回目**：第八十四回《难灭伽持圆大觉 法王成正体天然》
- **相关事件**：因僧人诽谤而发誓杀一万个和尚

## 📖 背景故事
灭法国国王是一个对佛教有偏见的君主。他因两年前被僧人诽谤，一怒之下发誓要杀一万个和尚，已经杀了九千九百九十六个，只差四个就完成目标。

当唐僧师徒来到灭法国时，国王正要杀他们凑数。孙悟空用计让国王和大臣们剃光头发，使他们认识到佛法的威力，最终国王放弃杀僧的念头，改国号为"钦法国"。

## 🎭 经典片段

### 1. **初见唐僧**
唐僧来到灭法国，国王态度凶狠：

```
国王："哈哈！又来了四个和尚！正好凑够一万！"
唐僧："陛下，为何要杀僧人？"
国王："哼！僧人最是可恶！朕要杀尽天下和尚！"
```

### 2. **诉说原因**
国王向唐僧诉说杀僧的原因：

```
国王："两年前，有个和尚诽谤朕，说朕要亡国！"
唐僧："陛下，一个和尚的话何必当真？"
国王："朕发誓要杀一万个和尚解恨！"
```

### 3. **孙悟空施法**
孙悟空夜间施法，剃光国王和大臣的头发：

```
孙悟空："陛下要杀和尚，俺老孙先让你们当和尚！"
（施展法术，剃光众人头发）
国王："啊！朕的头发！怎么回事？"
```

### 4. **醒悟忏悔**
国王认识到佛法的威力：

```
国王："这...这一定是佛祖显灵！朕知错了！"
唐僧："陛下现在明白佛法的威力了吧？"
国王："明白了！朕再也不杀僧人了！"
```

### 5. **改国号**
国王改国号为"钦法国"：

```
国王："朕要改国号为钦法国，尊崇佛法！"
孙悟空："陛下知错能改，善莫大焉！"
国王："多谢长老点醒！朕定当善待僧人！"
```

## 💡 性格特点

### 优点
- **知错能改**：发现错误后立即改正
- **从善如流**：愿意接受正确的建议
- **果断决策**：认识到错误后立即采取行动

### 特点
- **脾气暴躁**：容易因小事而动怒
- **固执己见**：一旦决定就很难改变
- **敬畏神灵**：对超自然力量心存敬畏

## 🔗 人物关系
- **诽谤僧**：导致他仇恨佛教的僧人
- **唐僧师徒**：帮助他认识错误的人
- **灭法国百姓**：受他统治的臣民

## 🌟 文化象征
灭法国国王代表了"知错能改"的美德：
- **敬畏之心**：对未知力量要保持敬畏
- **宽容大度**：不要因小事而仇恨整个群体
- **及时改正**：认识到错误要及时改正

## 💭 现代启示
1. **理性对待**：不要因个别现象而否定整体
2. **宽容待人**：对他人要保持宽容心态
3. **敬畏未知**：对不了解的事物要保持敬畏

灭法国国王的故事告诉我们，宽容和敬畏是重要的美德。
