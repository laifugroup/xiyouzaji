# 丞相张道源

## 📋 人物档案
- **姓名**：张道源
- **官职**：丞相
- **身份**：唐代官员
- **登场回目**：第十一回《还受生唐王遵善果 度孤魂萧瑀正空门》

## 📖 背景故事
张道源是唐太宗时期的宰相之一，在《西游记》中是一个次要但重要的官员角色。他在唐太宗决定举办水陆大会的朝廷辩论中扮演了一定角色。

作为朝廷重臣，张道源在重大决策中有着重要发言权。他的立场和态度对朝廷政策的制定有着重要影响。

## 🎭 经典片段

### 1. **朝廷议事**
唐太宗召集大臣商议举办水陆大会：

```
唐太宗："众位爱卿，朕欲举办水陆大会，超度亡灵，诸位意下如何？"
张道源："陛下圣明！此举有利于安抚民心，彰显陛下仁德。"
```

### 2. **支持佛教**
在傅奕反对佛教时，张道源表示支持：

```
傅奕："佛教乃夷狄之教，不应盛行！"
张道源："傅大人此言过矣。佛法劝人向善，有何不好？"
```

### 3. **调和矛盾**
在激烈的朝廷辩论中，张道源试图调和矛盾：

```
张道源："傅大人为国着想，其心可嘉。但佛教既已传入，强行禁止恐非上策。"
萧瑀："张相国言之有理，当以疏导为主。"
```

### 4. **参与决策**
作为宰相，参与重要国家决策：

```
唐太宗："张爱卿，水陆大会的具体事宜，就由你与萧相国共同负责。"
张道源："臣遵旨！定当尽心竭力，办好此事。"
```

## 💡 性格特点

### 优点
- **稳重持重**：在重大决策中保持冷静
- **善于协调**：能在不同意见间找到平衡
- **尽职尽责**：认真完成皇帝交办的任务

### 特点
- **政治智慧**：懂得在复杂局势中做出正确选择
- **顾全大局**：考虑问题从国家整体利益出发
- **中庸之道**：不偏激，寻求折中方案

## 🔗 人物关系
- **唐太宗**：服务的君主
- **萧瑀**：同为宰相，合作共事
- **傅奕**：政见不同的同僚

## 🌟 文化象征
张道源代表了传统政治中的"中庸之道"：
- **调和矛盾**：在不同意见间寻求平衡
- **稳重决策**：重大决策需要谨慎考虑
- **政治智慧**：懂得何时进，何时退

## 💭 现代启示
1. **协调能力**：在团队中善于调和不同意见
2. **大局观念**：考虑问题要从整体出发
3. **稳重行事**：重大决策需要深思熟虑

张道源的故事体现了传统政治中宰相的重要作用和中庸之道的价值。
