# 天竺国国王

## 📋 人物档案
- **身份**：天竺国国王
- **登场回目**：第九十三回《给孤园问古谈因 天竺国朝王遇偶》
- **相关事件**：女儿被玉兔精假冒，举办抛绣球招亲

## 📖 背景故事
天竺国国王是唐僧师徒西行路上遇到的最后一个外国君主。他的国家是天竺（古印度），是佛教的发源地。国王有一个美丽的女儿，但被月宫玉兔精假冒，真公主被藏在布金寺中。

当唐僧师徒来到天竺国时，正值假公主举办抛绣球招亲。唐僧被绣球打中，被迫成为驸马。后来孙悟空查明真相，救回真公主，国王感激不尽。

## 🎭 经典片段

### 1. **接见唐僧**
唐僧来到天竺国，国王热情接见：

```
国王："大唐高僧远道而来，真是敝国荣幸！"
唐僧："陛下过奖了，贫僧奉旨西天取经，路过贵国。"
国王："长老来得正好，朕的女儿正要抛绣球招亲！"
```

### 2. **抛绣球招亲**
假公主抛绣球，正好打中唐僧：

```
国王："哈哈！绣球打中圣僧，真是天作之合！"
唐僧："陛下！贫僧是出家人，不敢破戒！"
国王："出家人也可以还俗嘛！朕的女儿配得上你！"
```

### 3. **孙悟空识破**
孙悟空看出假公主的真面目：

```
孙悟空："陛下！你的女儿是妖怪变的！"
国王："胡说！朕的女儿怎会是妖怪？"
孙悟空："你看！"（施展法术让玉兔精现形）
国王："啊！果然是妖怪！朕的女儿在哪里？"
```

### 4. **父女团聚**
真公主被救回，父女团聚：

```
国王："女儿！你终于回来了！朕想死你了！"
公主："父王！女儿被妖怪关在布金寺三年了！"
国王："多谢孙长老！朕愿以江山相报！"
```

### 5. **送别唐僧**
国王热情送别唐僧师徒：

```
国王："圣僧保重！前面就是灵山了！"
唐僧："多谢陛下！贫僧定当完成取经使命。"
国王："愿佛祖保佑你们取经成功！"
```

## 💡 性格特点

### 优点
- **热情好客**：对远道而来的客人热情款待
- **爱女心切**：对女儿充满父爱
- **知恩图报**：对救命恩人充满感激

### 特点
- **重视传统**：遵循抛绣球招亲的传统
- **开明大度**：虽为国王但平易近人
- **虔诚信仰**：作为佛教发源地的君主，重视佛法

## 🔗 人物关系
- **天竺国公主**：心爱的女儿
- **玉兔精**：假冒公主的妖怪
- **唐僧师徒**：救命恩人

## 🌟 文化象征
天竺国国王代表了佛教文化的传承：
- **佛教圣地**：天竺是佛教的发源地
- **传统礼仪**：尊重和传承传统文化
- **父爱亲情**：父亲对女儿的深沉爱意

## 💭 现代启示
1. **文化传承**：重视和传承传统文化
2. **亲情可贵**：家人之间的感情最珍贵
