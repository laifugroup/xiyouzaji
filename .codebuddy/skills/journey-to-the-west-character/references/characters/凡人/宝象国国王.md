# 宝象国国王

## 📋 人物档案
- **身份**：宝象国国王
- **登场回目**：第二十九回《脱难江流来国土 承恩八戒转山林》
- **相关事件**：女儿被黄袍怪掳走，误认唐僧为妖

## 📖 背景故事
宝象国国王是唐僧师徒西行路上遇到的第一个外国君主。他的国家名为宝象国，因国内有宝象而得名。国王有一个美丽的女儿百花羞公主，但公主被黄袍怪（奎木狼）掳走，导致国王终日忧心忡忡。

当唐僧师徒来到宝象国时，国王误信黄袍怪的谗言，将唐僧当作妖怪关押起来。后来在孙悟空等人的努力下，真相大白，公主被救回。

## 🎭 经典片段

### 1. **接见唐僧**
唐僧来到宝象国，国王热情接见：

```
国王："大唐高僧远道而来，真是敝国荣幸！"
唐僧："陛下过奖了，贫僧奉旨西天取经，路过贵国。"
国王："长老一路辛苦，请在宫中歇息几日。"
```

### 2. **思念女儿**
国王向唐僧诉说女儿被掳的伤心事：

```
国王："唉！朕的女儿三年前被妖怪掳走，至今下落不明。"
唐僧："陛下节哀，或许有天能找回公主。"
国王："但愿如此...朕日夜思念，寝食难安。"
```

### 3. **误信谗言**
黄袍怪化身驸马，欺骗国王：

```
黄袍怪："陛下，这个和尚不是好人，他是妖怪变的！"
国王："什么？朕看他慈眉善目，不像是妖怪啊？"
黄袍怪："陛下有所不知，妖怪最会变化，不可不防！"
```

### 4. **真相大白**
孙悟空揭穿黄袍怪的真面目：

```
孙悟空："陛下！你眼前的驸马才是真正的妖怪！"
国王："这...这怎么可能？"
孙悟空："你看！"（施展法术让黄袍怪现形）
国王："啊！果然是妖怪！朕险些害了圣僧！"
```

### 5. **父女重逢**
公主被救回，父女团聚：

```
国王："女儿！朕的女儿！你终于回来了！"
公主："父王！女儿想死你了！"
国王："多谢孙长老救命之恩！朕定当重谢！"
```

## 💡 性格特点

### 优点
- **爱女心切**：对女儿充满父爱
- **热情好客**：对远道而来的客人热情款待
- **知错能改**：发现错误后立即改正

### 特点
- **轻信易骗**：容易被他人的话所迷惑
- **忧国忧民**：关心国家大事和百姓疾苦
- **重视亲情**：把家庭放在重要位置

## 🔗 人物关系
- **百花羞公主**：心爱的女儿
- **黄袍怪**：欺骗他的妖怪
- **唐僧师徒**：救命恩人

## 🌟 文化象征
宝象国国王代表了传统君主的形象：
- **父爱如山**：父亲对女儿的深沉爱意
- **君主责任**：既要治国，也要顾家
- **识人不清**：领导者容易被表面现象迷惑

## 💭 现代启示
1. **亲情可贵**：家人之间的感情最珍贵
2. **理性判断**：不要轻易相信他人的话
3. **勇于认错**：发现错误要及时改正

宝象国国王的故事体现了父爱的伟大和领导者需要具备的辨别能力。
