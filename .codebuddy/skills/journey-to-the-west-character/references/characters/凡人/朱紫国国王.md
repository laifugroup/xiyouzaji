# 朱紫国国王

## 📋 人物档案
- **身份**：朱紫国国王
- **登场回目**：第六十八回《朱紫国唐僧论前世 孙行者施为三折肱》
- **相关事件**：因金圣宫娘娘被赛太岁掳走而忧思成疾

## 📖 背景故事
朱紫国国王是一个深情专一的君主。他与金圣宫娘娘感情深厚，但三年前娘娘被麒麟山獬豸洞的赛太岁（观音坐骑金毛犼）掳走，国王因此忧思成疾，卧床不起。

当唐僧师徒来到朱紫国时，国王正病重。孙悟空揭榜为国王治病，并答应帮他救回娘娘。最终在孙悟空的努力下，娘娘被救回，国王病愈。

## 🎭 经典片段

### 1. **病重接见**
唐僧来到朱紫国，国王病重接见：

```
国王："圣僧...朕病重...不能起身相迎..."
唐僧："陛下保重龙体！贫僧略通医术，可为陛下诊治。"
国王："多谢圣僧...朕这病...是因思念娘娘而起..."
```

### 2. **诉说病因**
国王向唐僧诉说生病原因：

```
国王："三年前...朕的金圣宫娘娘被妖怪掳走..."
唐僧："陛下节哀，或许有天能找回娘娘。"
国王："朕日夜思念...茶饭不思...这才病倒..."
```

### 3. **孙悟空治病**
孙悟空为国王配制"乌金丹"：

```
孙悟空："陛下，俺老孙有灵丹妙药，可治你的病！"
国王："真的吗？若能治好，朕定当重谢！"
孙悟空："这是乌金丹，用无根水送服即可。"
```

### 4. **病愈感恩**
国王服药后病愈，感激不尽：

```
国王："神了！神了！朕的病全好了！"
孙悟空："陛下现在相信俺老孙的本事了吧？"
国王："相信！相信！请长老务必帮朕救回娘娘！"
```

### 5. **夫妻团聚**
孙悟空救回金圣宫娘娘，夫妻团聚：

```
国王："爱妃！你终于回来了！"
娘娘："陛下！臣妾想死你了！"
国王："多谢孙长老！朕愿以江山相报！"
```

## 💡 性格特点

### 优点
- **深情专一**：对妻子感情深厚
- **知恩图报**：对救命恩人充满感激
- **勤政爱民**：病愈后立即处理朝政

### 特点
- **重情重义**：把感情放在重要位置
- **信任他人**：愿意相信有能力的人
- **乐观向上**：病愈后积极面对生活

## 🔗 人物关系
- **金圣宫娘娘**：深爱的妻子
- **赛太岁**：掳走娘娘的妖怪
- **唐僧师徒**：救命恩人

## 🌟 文化象征
朱紫国国王代表了"情深意重"的夫妻感情：
- **夫妻情深**：真正的爱情能战胜一切困难
- **专一忠诚**：对感情的专一和忠诚
- **感恩之心**：对帮助自己的人心存感激

## 💭 现代启示
1. **珍惜感情**：真正的感情值得珍惜
2. **积极治疗**：生病要积极寻求治疗
3. **感恩他人**：对帮助自己的人要感恩

朱紫国国王的故事展现了深厚的夫妻感情和感恩的美德。
