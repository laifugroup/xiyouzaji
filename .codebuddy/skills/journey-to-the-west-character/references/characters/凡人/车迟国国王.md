# 车迟国国王

## 📋 人物档案
- **身份**：车迟国国王
- **登场回目**：第四十四回《法身元运逢车力 心正妖邪度脊关》
- **相关事件**：尊崇虎力、鹿力、羊力三位大仙，迫害佛教徒

## 📖 背景故事
车迟国国王是一个被妖怪蒙蔽的君主。他的国家遭遇大旱时，虎力大仙、鹿力大仙、羊力大仙三个妖怪前来求雨成功，国王便尊崇他们为国师，在全国推行道教，迫害佛教徒。

当唐僧师徒来到车迟国时，国王起初对他们十分冷淡。后来在孙悟空与三位大仙的斗法过程中，国王逐渐认识到真相，最终改邪归正。

## 🎭 经典片段

### 1. **初见唐僧**
唐僧来到车迟国，国王态度冷淡：

```
国王："又是和尚？朕的国家不欢迎和尚！"
唐僧："陛下，贫僧奉旨西天取经，路过贵国..."
国王："取经？哼！我国三位大仙才是真神仙！"
```

### 2. **尊崇妖道**
国王向唐僧炫耀三位大仙的本领：

```
国王："朕的三位国师能呼风唤雨，你们和尚能做什么？"
唐僧："陛下，佛法无边，不可轻视。"
国王："哼！你们和尚只会念经，毫无用处！"
```

### 3. **观看斗法**
国王观看孙悟空与三位大仙斗法：

```
国王："今日朕要看看，到底是你们的佛法厉害，还是朕的道法厉害！"
孙悟空："陛下请看！"（展示神通）
国王："这...这和尚果然有些本事！"
```

### 4. **逐渐醒悟**
在斗法过程中，国王开始怀疑三位大仙：

```
国王："三位国师怎么连连败退？莫非..."
孙悟空："陛下明鉴！他们根本不是什么神仙，而是妖怪！"
国王："啊！朕被他们骗了这么多年！"
```

### 5. **改邪归正**
三位大仙现出原形，国王彻底醒悟：

```
国王："原来真是妖怪！朕真是瞎了眼！"
孙悟空："陛下现在明白也不晚。"
国王："多谢长老点醒！朕定当重兴佛教，善待僧人！"
```
