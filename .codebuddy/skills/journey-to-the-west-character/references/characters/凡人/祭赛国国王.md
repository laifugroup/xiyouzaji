# 祭赛国国王

## 📋 人物档案
- **身份**：祭赛国国王
- **登场回目**：第六十二回《涤垢洗心惟扫塔 缚魔归正乃修身》
- **相关事件**：因佛宝舍利子丢失而怪罪僧人

## 📖 背景故事
祭赛国国王是一个重视佛教但容易误判的君主。他的国家原本因有佛宝舍利子而祥光普照，四方来朝。但舍利子被万圣龙王和九头虫盗走后，国王误以为是本国僧人监守自盗，因而迫害僧人。

当唐僧师徒来到祭赛国时，发现金光寺的僧人正在受罚。孙悟空查明真相后，帮助找回了佛宝，国王这才明白错怪了僧人。

## 🎭 经典片段

### 1. **初见唐僧**
唐僧来到祭赛国，国王态度冷淡：

```
国王："又是和尚？朕的国家不欢迎和尚！"
唐僧："陛下，为何如此对待僧人？"
国王："哼！朕的佛宝被你们和尚偷走了！"
```

### 2. **诉说冤屈**
国王向唐僧诉说佛宝丢失的经过：

```
国王："三年前，朕的佛宝舍利子突然不见了！"
唐僧："陛下如何确定是僧人所为？"
国王："除了他们还有谁？佛宝一直在金光寺供奉！"
```

### 3. **观看扫塔**
国王观看孙悟空扫塔寻宝：

```
国王："这个毛脸和尚说要扫塔寻宝，真是可笑！"
孙悟空："陛下请看！宝塔顶上果然有妖气！"
国王："什么？难道真有妖怪作祟？"
```

### 4. **真相大白**
孙悟空查明真相，国王恍然大悟：

```
孙悟空："陛下！佛宝是被碧波潭的万圣龙王盗走的！"
国王："啊！朕错怪了僧人！真是罪过！"
唐僧："陛下现在明白也不晚。"
```

### 5. **重兴佛教**
佛宝找回后，国王重兴佛教：

```
国王："朕要重修金光寺，善待僧人，弥补过错！"
孙悟空："陛下知错能改，善莫大焉！"
国王："多谢长老指点迷津！"
```
