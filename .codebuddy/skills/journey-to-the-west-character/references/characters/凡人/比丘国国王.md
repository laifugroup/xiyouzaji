# 比丘国国王

## 📋 人物档案
- **身份**：比丘国国王
- **登场回目**：第七十八回《比丘怜子遣阴神 金殿识魔谈道德》
- **相关事件**：被白鹿精和白面狐狸迷惑，欲用小儿心肝做药引

## 📖 背景故事
比丘国国王是一个被妖怪严重迷惑的君主。他被寿星坐骑白鹿精变化的国丈和狐狸精变化的美后所迷惑，听信他们的谗言，欲用一千一百一十一个小儿的心肝做药引，以求长生不老。

当唐僧师徒来到比丘国时，发现满城百姓都在哀哭。孙悟空查明真相后，揭穿了妖怪的真面目，救下了所有小儿。

## 🎭 经典片段

### 1. **初见唐僧**
唐僧来到比丘国，国王态度冷淡：

```
国王："和尚？朕现在没空理会和尚！"
唐僧："陛下，贫僧见城中百姓哀哭，不知为何？"
国王："与你无关！朕要炼制长生不老药！"
```

### 2. **诉说药方**
国王向唐僧炫耀"长生不老药"：

```
国王："朕的国丈献上仙方，用小儿心肝做引，可长生不老！"
唐僧："陛下！此乃伤天害理之事，万万不可！"
国王："哼！你懂什么？长生不老才是大事！"
```

### 3. **孙悟空揭穿**
孙悟空在朝堂上揭穿妖怪真面目：

```
孙悟空："陛下！你的国丈和美后都是妖怪变的！"
国王："胡说！国丈是神仙，怎会是妖怪？"
孙悟空："你看！"（施展法术让妖怪现形）
国王："啊！果然是妖怪！朕被他们骗了！"
```

### 4. **醒悟忏悔**
国王认识到自己的错误：

```
国王："朕真是昏了头！竟相信妖怪的话！"
唐僧："陛下现在明白也不晚。"
国王："朕要立即释放所有小儿，向百姓谢罪！"
```

### 5. **重振朝纲**
妖怪被收服后，国王重振朝纲：

```
国王："多谢长老救命之恩！朕定当勤政爱民，弥补过错！"
孙悟空："陛下知错能改，善莫大焉！"
国王："朕再也不会相信长生不老的鬼话了！"
```

## 💡 性格特点

### 优点
- **知错能改**：发现错误后立即改正
- **重视民生**：醒悟后立即关心百姓疾苦
- **从善如流**：愿意接受正确的建议

### 特点
- **贪生怕死**：过于追求长生不老
- **轻信易骗**：容易被花言巧语所迷惑
- **转变迅速**：认识到错误后立即改正

## 🔗 人物关系
- **白鹿精**：欺骗他的妖怪国丈
- **白面狐狸**：迷惑他的妖怪美后
- **唐僧师徒**：救命恩人
- **比丘国百姓**：受他统治的臣民

## 🌟 文化象征
比丘国国王代表了"贪生怕死"的人性弱点：
- **长生诱惑**：对永生的渴望容易让人迷失
- **理性判断**：面对诱惑要保持理性
- **为民着想**：君主应该把百姓利益放在首位

## 💭 现代启示
1. **理性思考**：不要被不切实际的承诺所迷惑
2. **珍视生命**：任何人的生命都值得尊重
3. **知错能改**：发现错误要及时改正

比丘国国王的故事警示我们，追求长生不老往往会导致更大的灾难。
