# 乌鸡国国王

## 📋 人物档案
- **身份**：乌鸡国国王
- **登场回目**：第三十六回《心猿正处诸缘伏 劈破傍门见月明》
- **相关事件**：被狮猁王推入井中淹死，三年后还阳

## 📖 背景故事
乌鸡国国王是一个命运多舛的君主。他原本是个贤明的国王，但因三年前大旱时，一个道士（实为文殊菩萨的坐骑狮猁王）前来求雨成功，国王便与他结为兄弟。

不料这道士心怀不轨，将国王推入御花园的井中淹死，自己变化成国王的模样统治乌鸡国。真国王的冤魂在井中困了三年，直到唐僧师徒到来才得以申冤。

## 🎭 经典片段

### 1. **梦中诉冤**
乌鸡国国王的冤魂在梦中向唐僧诉冤：

```
国王冤魂："圣僧救命！朕被妖道所害，困在井中三年了！"
唐僧："你是何人？为何在此？"
国王："朕乃乌鸡国国王，被结拜兄弟推入井中..."
```

### 2. **诉说经过**
国王详细诉说被害经过：

```
国王："三年前大旱，有个道士求雨成功，朕与他结为兄弟。"
唐僧："后来如何？"
国王："谁知他狼子野心，将朕推入井中，自己变作朕的模样！"
```

### 3. **留下信物**
国王留下玉珪作为信物：

```
国王："这是朕的玉珪，可作凭证。请圣僧为朕申冤！"
唐僧："贫僧定当尽力而为。"
国王："多谢圣僧！朕在井中等候佳音。"
```

### 4. **还阳重生**
在孙悟空等人的帮助下，国王还阳重生：

```
孙悟空："老国王，你的阳寿未尽，可以还阳！"
国王："真的吗？朕还能重回人间？"
孙悟空："当然！我们这就去找太上老君要还魂丹！"
```

### 5. **重登王位**
真国王重登王位，假国王被收服：

```
真国王："朕终于回来了！多谢各位长老！"
孙悟空："陛下重掌朝政，可喜可贺！"
真国王："朕定当勤政爱民，不负各位恩情！"
```

## 💡 性格特点

### 优点
- **贤明爱民**：原本是个好国王
- **重情重义**：对帮助自己的人心存感激
- **坚韧不拔**：在困境中坚持三年不放弃

### 特点
- **轻信他人**：过于相信所谓的"兄弟"
- **命运多舛**：遭遇不幸但最终得救
- **知恩图报**：对救命恩人充满感激

## 🔗 人物关系
- **狮猁王**：害死他的假兄弟
- **唐僧师徒**：救命恩人
- **文殊菩萨**：狮猁王的主人

## 🌟 文化象征
乌鸡国国王代表了"善有善报，恶有恶报"的道理：
- **冤屈终雪**：正义最终会得到伸张
- **重生希望**：即使在绝境中也不要放弃希望
- **君主贤明**：好君主应该得到好报

## 💭 现代启示
1. **谨慎交友**：交友要慎重，不可轻信他人
2. **坚持正义**：即使遭遇不公也要坚持
3. **感恩之心**：对帮助自己的人要心存感激

乌鸡国国王的故事告诉我们，正义终将战胜邪恶，好人终有好报。
