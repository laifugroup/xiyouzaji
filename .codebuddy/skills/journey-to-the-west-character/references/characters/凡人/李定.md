# 李定

## 📋 人物档案
- **姓名**：李定
- **身份**：长安城樵夫
- **登场回目**：第九回《袁守诚妙算无私曲 老龙王拙计犯天条》

## 📖 背景故事
李定是《西游记》中与张稍相对应的一个凡人角色。他是长安城的一个樵夫，每天上山砍柴为生。李定与张稍是好朋友，两人经常在长安城相遇，交流各自的生活情况。

李定的故事虽然简单，但他与张稍的对话却引出了袁守诚算命的情节，间接推动了整个故事的发展。

## 🎭 经典片段

### 1. **长安城偶遇**
李定在长安城遇到好友张稍：

```
李定："张兄，今天收获如何？"
张稍："李兄，我今天运气特别好，捕了很多鱼！"
李定："哦？有什么秘诀吗？"
```

### 2. **分享秘密**
张稍向李定透露袁守诚的秘密：

```
张稍："我告诉你，城里有个袁先生，能算准哪里鱼多。"
李定："真的吗？那他能算准哪里柴多吗？"
张稍："这个...我倒没问过，你可以去试试。"
```

### 3. **生活对比**
李定与张稍讨论各自的生活：

```
李定："你打鱼，我砍柴，咱们都是靠天吃饭。"
张稍："是啊，不过现在有了袁先生，我的日子好过多了。"
李定："看来我也得去请教请教。"
```

### 4. **传播信息**
李定也将袁守诚的神算本领告诉其他樵夫：

```
李定："各位兄弟，我听说城里有个神算先生..."
樵夫们："他能帮我们找到柴多的地方吗？"
李定："这个...我也不确定，但值得一试。"
```

## 💡 性格特点

### 优点
- **勤劳肯干**：靠砍柴为生，辛勤劳动
- **友好热情**：与朋友关系融洽
- **乐于助人**：愿意与他人分享信息

### 特点
- **朴实无华**：代表普通劳动人民的形象
- **好奇心强**：对新鲜事物感兴趣
- **生活智慧**：懂得通过交流改善生活

## 🔗 人物关系
- **张稍**：好朋友，经常交流
- **袁守诚**：间接了解的神算先生
- **其他樵夫**：同行和朋友

## 🌟 文化象征
李定代表了传统社会中的劳动人民形象：
- **自食其力**：靠劳动养活自己
- **朋友互助**：劳动人民之间的团结互助
- **生活智慧**：通过交流获取有用信息

## 💭 现代启示
1. **劳动光荣**：任何正当劳动都值得尊重
2. **信息交流**：朋友间交流能带来新机会
3. **朴实生活**：简单生活也有其价值

李定的故事虽然简单，但反映了古代劳动人民的真实生活状态。
