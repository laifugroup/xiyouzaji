# 五方揭谛

## 人物图谱

### 基本信息图谱
- **身份定位**：佛界护法神，守护佛法五方
- **所属体系**：西天佛界，护法神体系
- **首次出场**：第十五回《蛇盘山诸神暗佑 鹰愁涧意马收缰》
- **形象特征**：五位护法神将，各守一方，威严庄重

- **修行历程**：从普通护法晋升为五方揭谛
- **重要转折**：受命暗中保护取经团队
