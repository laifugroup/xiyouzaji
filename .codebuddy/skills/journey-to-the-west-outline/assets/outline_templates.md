# 《西游记》大纲设计模板

## 整体结构模板

### 五大阶段模板
```
1. 序幕：孙悟空出世与大闹天宫（第1-7回）
   - 石猴出世，拜师学艺
   - 大闹天宫，被压五行山

2. 引子：取经任务缘起（第8-12回）
   - 如来传经，观音寻僧
   - 唐僧受命，启程西行

3. 开端：取经团队组建（第13-22回）
   - 收悟空，得龙马
   - 降八戒，服沙僧

4. 主体：西行取经历程（第23-99回）
   - 八十一难，重重考验
   - 降妖除魔，不断成长

5. 结局：取经圆满成功（第100回）
   - 到达灵山，取得真经
   - 返回东土，修成正果
```

## 章节标题与总结模板

### 标准章节模板
```
标题：[七言对仗句]
总结：[2-3句核心情节概括]

示例：
标题：灵根育孕源流出 心性修持大道生
总结：花果山石猴出世，发现水帘洞成为美猴王。为求长生出海拜师，得名孙悟空开始修行之路。
```

### 百回结构模板
```
第1-7回：孙悟空出世与大闹天宫
第8-12回：取经任务缘起
第13-22回：取经团队组建
第23-49回：前期八十一难
第50-74回：中期磨难考验
第75-99回：后期内心觉悟
第100回：取经圆满成功
```
