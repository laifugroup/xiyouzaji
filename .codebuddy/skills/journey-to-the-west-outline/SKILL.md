# 《西游记》大纲设计技能

## 技能用途

此技能旨在提供《西游记》简洁明了的小说大纲，快速呈现故事的整体情节框架和章节概要。适用于：

- **快速了解**：快速掌握《西游记》的整体故事框架和章节安排
- **创作参考**：为西游题材创作提供简洁的情节结构参考
- **教学入门**：为语文教学、文学鉴赏提供简明的大纲概览
- **改编基础**：为现代改编提供核心情节骨架

## 何时使用此技能

当用户需要：
- 快速了解《西游记》的整体故事框架
- 查询简单的章节标题和核心情节
- 获取2-3句的章节内容总结
- 了解取经路线的主要阶段划分
- 掌握八十一难的基本分布情况

## 注意事项

1. **简洁明了**：所有输出力求简洁，整体情节概览和章节总结均控制在2-3句内
2. **核心突出**：聚焦故事核心情节，避免细节描述
3. **结构清晰**：保持整体框架的清晰性和逻辑性
4. **实用导向**：便于用户快速了解故事框架和章节概要
5. **扩展可选**：用户可通过参考资料获取更详细的信息
